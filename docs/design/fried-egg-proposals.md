## 方案A：极简煎蛋 🥚

### 造型描述

一颗平放在桌上的煎蛋，就是这么直白。

外壳是**不规则的圆形**——不是几何意义上的正圆，而是煎蛋蛋白那种自然流淌后凝固的边缘，有轻微的波浪和凸起，每一颗都像是刚从锅里铲出来的。边缘厚度渐变，中心最厚（因为藏了屏幕和主板），边缘薄到约2mm，像真正蛋白的薄脆边。

正面中心是一块 **1.28寸圆形LCD**，完美嵌入壳体，屏幕玻璃与外壳表面几乎齐平，过渡处有一圈微微凸起的环（像蛋黄和蛋白的自然分界线）。屏幕亮起时就是金黄色的蛋黄，灭屏时是低调的黑色圆点。

外壳表面是**哑光磨砂质感**，触感温润，像婴儿肌肤（这就是"想让人摸一下"的关键——不是光面塑料的廉价感，而是细腻磨砂的亲肤感）。颜色是偏暖的**奶白色**（RAL 9010 纯白偏黄），不是冷白。

背面平整，中心区域嵌入一块 **圆形钕磁铁**（Ø20mm），可以吸附在冰箱、白板、金属桌面支架上。背面有一圈硅胶防滑垫脚（3个小圆点），平放时不会滑动。背面中心有产品logo的微凹雕刻。

USB-C充电/数据口藏在边缘最厚处，开口极小，不破坏煎蛋的整体感。

没有按钮。开机靠轻拍屏幕（触摸唤醒）或通电自启。

**共2件：**

1. **上壳（蛋白正面）**
   - 不规则圆形，中心开Ø34mm圆孔（放LCD模组）
   - 内侧有4个M2螺丝柱（与下壳对位）
   - 内侧有PCB定位卡槽

   - 层高0.12mm，正面朝下打印以获得光滑表面

2. **下壳（背面盖板）**
   - 扁平圆形，与上壳配合
   - 中心有磁铁安装凹槽（Ø20.5mm × 2.2mm深）
   - 3个硅胶垫脚安装位
   - 侧面有USB-C开口
   - 4个M2螺丝孔（与上壳对位）

### 与未来蛋蛋机器人的关系

方案A是**纯粹的Display+Core一体单元**。它是蛋蛋机器人的"脸"。

进化路径：
- **现在**：独立的煎蛋小设备，显示表情、时间、通知
- **下一步**：加一个底座（→ 变成方案B），获得更好的音频和续航
- **再下一步**：底座进化成蛋形身体，煎蛋变成"肚子上的屏幕"
- **最终形态**：煎蛋嵌入蛋蛋机器人的腹部，成为它的表情/信息窗口

关键设计约束：方案A的外形尺寸和屏幕位置要与未来蛋蛋机器人腹部的嵌入位兼容。

## 方案B：煎蛋+底座 🥚🏠

### 造型描述

想象一颗煎蛋斜靠在一个小蛋杯上——这就是方案B。

**煎蛋主体**和方案A基本相同（不规则圆形外壳+圆形屏幕），但更薄更轻，因为Core模块和电池都搬到了底座里。煎蛋部分只保留LCD屏幕和最小排线接口。

**底座**是一个矮胖的圆柱体，顶部有一个15°倾斜的凹面，煎蛋斜靠在上面，屏幕自然朝向坐在桌前的用户。底座直径约55mm，高约30mm，像一个可爱的小蘑菇墩。

底座顶面的凹槽里有一组 **RWC Bus 8pin磁吸pogo pin**——煎蛋往上一放就自动对位吸附、供电、通信。拿起来也很轻松，磁力恰到好处（像AirPods放进充电盒的感觉）。

底座正面（面向用户的方向）有一排细密的**扬声器出音孔**，排列成微笑的弧线（一个小彩蛋：出音孔的排列就是一个笑脸）。

底座的材质和煎蛋外壳一致——奶白色哑光磨砂。底座底部有一整圈硅胶防滑垫，分量略重（因为有电池），放在桌上很稳。

底座背面有USB-C口和一个实体小按钮（reset/配对用，凹陷设计，需要用指甲按）。

煎蛋从底座上取下后，可以靠内置的纽扣电池（CR2032）维持低功耗蓝牙运行约24小时——拿着到处走也行，贴在冰箱上也行，但没有WiFi和扬声器。

### 与未来蛋蛋机器人的关系

方案B的底座就是蛋蛋机器人**身体的原型**。

进化路径：
- **现在**：煎蛋+底座，桌面智能伴侣
- **下一步**：底座变高变椭圆 → 蛋形身体，煎蛋嵌入正面
- **再下一步**：身体顶部加触角模块（天线/LED），底部加轮子或脚
- **最终形态**：完整蛋蛋机器人，煎蛋部分是肚子屏幕，底座进化成身体

关键：底座的RWC Bus接口规范和煎蛋主体的磁吸对位方式要成为标准，未来所有模块都用同样的连接。

## 方案C：煎蛋进化体 🥚⚡

### 造型描述

这是给极客和早期adopter准备的——一颗煎蛋，但你能看出它**不只是一颗煎蛋**。

整体造型基于方案A的不规则煎蛋，但做了以下进化：

**外壳**不再是纯粹的有机煎蛋形——在不规则蛋白的边缘，底部、左侧、右侧各有一个**几何化的凹口**，露出金属质感的RWC Bus磁吸接口。这三个凹口打破了煎蛋的纯有机形态，暗示"这里可以长出什么"。凹口周围有一圈极细的**赛博绿色环形LED**（每个接口一圈，共3圈），设备运行时微微发光，像电路在蛋白下面若隐若现。

外壳表面不是纯白——是**半透明的白色**（像冻奶/牛奶冰），隐约能透出内部PCB的绿色和元器件的轮廓。这种"看得见内脏"的设计既有赛博朋克感，又让极客觉得亲切（"这不是消费电子，这是开发板"）。

正面中心依然是1.28寸圆形屏幕（蛋黄），但屏幕外圈多了一环**不锈钢装饰圈**（CNC或冲压），为蛋黄加了一个"画框"，提升质感。

背面是平面，中心有一个**大面积散热铜箔贴片**（既实用又好看），周围有4颗暴露在外的螺丝（不藏了，工业风）。

三个RWC Bus扩展口的位置：
- **底部**：主扩展口，用于连接底座/身体模块
- **左侧**：辅助口，用于Audio模块（未来的"左耳"）
- **右侧**：辅助口，用于传感器等（未来的"右耳"）

每个接口在没有连接模块时，有一个**磁吸保护盖**（小圆片，同色半透明白），防灰防误触，取下保护盖就能接模块。

| 整体最大直径 | ~85mm（比方案A大一圈，为接口留空间） |
| 中心最大厚度 | 15mm |
| 边缘最薄处 | 4mm（比方案A厚，因为走线需要） |
| 屏幕可视区域 | Ø32.4mm（1.28寸） |
| 扩展接口凹口 | 各约12mm宽 × 8mm深 |
| 重量 | ~55g |

### 屏幕选型

主选：**GC9A01 1.28寸圆形IPS LCD**（240×240，¥15-18）

可选升级：**1.69寸圆形LCD** 或 **1.28寸圆形AMOLED**（如有货源，AMOLED的纯黑背景让蛋黄效果更惊艳，但成本翻倍到¥40+）

   - 小圆片，内嵌Ø3mm小磁铁

### 与未来蛋蛋机器人的关系

方案C是最明确的**进化起点**——它身上已经长出了"接口"，在对用户说："给我接上更多东西吧。"

进化路径：
- **现在**：三接口煎蛋，独立运行
- **第二步**：底部接口插上Audio底座 → 有了声音（方案B底座的进化版）
- **第三步**：左右接口插上小圆球模块 → 蛋蛋长了"耳朵"（传感器/装饰）
- **第四步**：底座拉长成蛋形身体，煎蛋嵌入腹部，耳朵变成触角
- **第五步**：身体底部接Servo模块 → 蛋蛋有了脚，可以动了
- **最终形态**：完整的蛋蛋机器人，而最初买的煎蛋进化体还在——它是肚子

方案C的哲学：**你买的不是一个成品，你买的是一个生命的第一个细胞。**

## 美羊羊的建议 🎀

三个方案不是三选一——它们是**产品线的三个SKU**：

1. **先做方案A**。成本最低、打印最简单、最快出原型。用它验证煎蛋造型的可爱程度和基本功能。
2. **方案B是主力产品**。有声音、有底座、有分离设计，功能完整，适合大多数用户。
3. **方案C是开发者版/限量版**。给硬核用户和共创社区，让他们帮你设计扩展模块。

三者共享同一块1.28寸圆形LCD、同一颗ESP32-S3、同一套固件。差异只在外壳和扩展性。

最重要的事：**先让煎蛋的表情动起来**。哪怕硬件还没定，先用屏幕驱动代码做一组可爱的眨眼、微笑、打哈欠动画。当人们第一次看到蛋黄亮起来、对他们眨眼的时候——他们就想买了。
