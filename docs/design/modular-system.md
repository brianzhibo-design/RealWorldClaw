## 核心理念

**RealWorldClaw不是一个产品，是一套AI硬件的乐高系统。**

- 标准化模块（电子积木）= 乐高的砖块
- 3D打印结构件 = 乐高的底板和特殊件
- 参考设计 = 乐高的搭建图纸
- 用户自由组合 = 无限可能

---

## 模块接口标准（Module Interface Standard）

### 物理接口
所有模块统一为一种连接方式，让任意两个模块可以对接。

**待定设计要素：**
- 机械尺寸：模块的基本网格单位（如20mm×20mm的倍数？）
- 连接方式：磁吸？卡扣？螺丝？Pogo Pin？
- 接口位置：顶部？底部？四面？

### 电气接口
- 电源总线：统一电压（3.3V? 5V?）
- 通信总线：I2C为主（简单、多设备），SPI可选（高带宽）
- 每个模块自带ID（EEPROM或电阻编码），主控自动识别

### 通信协议
- 模块间：I2C总线
- 对外：Wi-Fi/BLE（主控模块负责）
- 与AI Agent：MQTT

## 核心模块清单（Phase 1）

### 必须有的（最小可用系统）

| 模块 | 功能 | 预估成本 | 标准件采购 |
|------|------|---------|-----------|
| **Core 主控模块** | ESP32-S3, Wi-Fi, BLE, USB-C | ¥20-25 | ✅ |
| **Power 电源模块** | 锂电池+充电管理+电源分配 | ¥15-20 | ✅ |
| **Display 显示模块** | OLED/LCD屏，显示表情/信息 | ¥10-15 | ✅ |

这三个模块组合 = 一个最基础的AI终端，有脸、有网、有电。

### 拓展模块（按需选购）

| 模块 | 功能 | 预估成本 |
|------|------|---------|
| **Audio 音频模块** | 麦克风+喇叭，语音交互 | ¥15 |
| **Sensor 传感器模块** | 温湿度/光照/空气质量 | ¥10-15 |
| **Motor 运动模块** | 舵机/电机驱动，让它动起来 | ¥15-20 |
| **Camera 摄像头模块** | 视觉感知 | ¥15 |
| **LED 灯光模块** | RGB灯效，氛围/状态指示 | ¥5-10 |
| **IMU 姿态模块** | 加速度计+陀螺仪 | ¥5 |
| **GPS 定位模块** | 户外定位 | ¥15 |
| **Relay 继电器模块** | 控制外部设备 | ¥5 |

### 结构件（3D打印，社区设计）

这些不是标准模块，而是千变万化的外壳/骨架：
- 桌面底座（容纳Core+Power+Display）
- 人形骨架（容纳多个Motor模块）
- 六足底盘（容纳6个Motor模块）
- 小车底盘（容纳2个Motor模块）
- 墙挂壳体（容纳Core+Sensor）
- 可穿戴外壳（容纳Core+Display）
- ...任何人都可以设计新的结构件

| 参考设计 | 模块组合 | 结构件 | 总成本 |

| 桌面AI助手 | Core+Power+Display+Audio | 桌面底座 | ~¥70 |

| 桌面人偶 | Core+Power+Display+Motor×4 | 人形骨架 | ~¥120 |
| 环境监测站 | Core+Power+Sensor+Display | 墙挂壳体 | ~¥60 |

## 商业模式的影响

这套系统改变了商业模式：

### 平台卖什么
1. **标准模块套装**（电子件，标准件采购/代采购）
2. **打印服务**（结构件，制造者网络）
3. **参考设计**（免费开源，引流用）
4. **定制设计服务**（用户要独特的外壳，设计师接单）

### 制造者做什么
- **打印者**：打印结构件（外壳、骨架、底座）
- **组装者**：帮用户把模块+结构件组装成成品
- **设计者**：设计新的结构件/参考设计

### 冷启动
1. 先定义模块接口标准
2. 出3个核心模块（Core+Power+Display）
3. 出2-3个参考设计（桌面助手+六足蜘蛛）
4. 用我们自己的P2S打印第一批结构件
5. 在极客社区推广
