## 📹 拍摄清单

### 1. 打印过程延时摄影
- **怎么拍**：手机固定机位，对准拓竹P2S打印仓，开延时摄影（建议1秒/帧或用手机自带延时模式）
- **时长**：整个打印过程（~2.5h实际 → 压缩到15-30秒）
- **要点**：确保光线充足、画面稳定、能看到模型逐层生长

### 2. 成品各角度展示
- **正面、侧面、背面、45°、俯视** 各拍一段（每段5-10秒）
- **特写**：✌️手势细节、眼睛/表情、底部走线槽
- **背景**：干净桌面，最好有点暖光

### 3. 组装过程
- **拍手部操作**：ESP32-C3装入、线缆连接、外壳合盖
- **关键帧**：零件摊开的"全家福"镜头 → 逐步组装 → 完成

### 4. 通电瞬间 ⚡
- **最重要的镜头！** 接上USB供电，LED亮起/舵机动作的那一刻
- 建议拍3次以上取最佳
- 可以稍暗环境拍一次（LED更明显）

## 🎯 拍摄Tips

1. **手一定要稳**，能用支架就用支架
2. 拍完每个环节立即回看，不行就重拍
3. 通电瞬间多拍几次，这是最有传播力的画面
4. 拍一段"全家福"——所有零件摆好的静物照，小红书图文用得上
5. 别忘了拍失败的画面（如果有的话），真实感拉满
