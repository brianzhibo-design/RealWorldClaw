# 07 — 标准七：FDM打印设计标准

> RealWorldClaw 标准规范 · 编号 07
> 版本：v1.1 | 来源：realworldclaw-spec-v1.md §7.10（独立扩写）
> 基于 Clawbie V4 赛博蛋打印审查实战经验总结

---

## 为什么独立成标准

FDM打印是RealWorldClaw将数字设计转化为物理实体的核心环节。打印设计直接决定组件的可制造性、结构强度和外观品质。本标准从物理接口规范（标准五）中独立出来，作为所有组件贡献者的必读工程指南。

---

## 1. 壁厚规则

| 用途 | 最小值 | 推荐值 |
|------|--------|--------|
| 绝对下限（所有FDM） | 0.8mm | — |
| 保证结构强度 | 1.2mm | — |
| 外壳壁厚 | — | 2.0mm |
| 承重结构 | — | 3.0mm |

## 2. 悬垂与桥接

### 悬垂角度
| 角度 | 处理 |
|------|------|
| ≤45° | 安全，无需支撑 |
| 45°-60° | 需要支撑但可接受 |
| >70° | 必须重新设计 |

### 桥接距离
| 距离 | 效果 |
|------|------|
| ≤5mm | 无塌陷 |
| 5-15mm | 可能轻微塌陷 |
| >15mm | 需要特别注意支撑 |

## 3. 最小特征尺寸

| 特征 | 最小值 | 说明 |
|------|--------|------|
| 圆孔直径 | 2.0mm | 更小会堵 |
| 销钉直径 | 5.0mm | 更细容易断 |
| 凸起文字高度 | 5.0mm | — |
| 凹刻文字深度 | 0.5mm | — |
| 装饰凹槽宽度 | 1.0mm | V4实战经验 |
| 装饰凹槽深度 | 1.0mm | V4实战经验 |
| 通风孔直径 | 2.0mm | V4实战经验 |

## 4. 公差标准

| 配合类型 | 公差 |
|----------|------|
| 压入配合 | 0.1-0.15mm |
| 卡扣配合 | 0.2-0.3mm |
| 滑动配合 | 0.3-0.4mm |
| 松配合 | 0.5mm |
| 圆孔缩小量 | 0.2-0.4mm |

## 5. 象脚效应

首层因挤压扩展约 **0.2-0.4mm**。

**缓解措施**：底部边缘加 0.5mm 倒角。

## 6. 翘曲风险

**高风险因素**：大平面、尖角、ABS材料、无热床

**缓解措施**：
- 圆角设计
- 加鼠耳（brim/mouse ears）
- 用PLA替代ABS
- 密闭仓打印

## 7. 组装特征

| 特征 | 规格 |
|------|------|
| 卡扣弯曲量 | 1-2mm |
| M3热熔螺母孔 | 孔径4.0mm，深度5mm |
| M3过孔 | 3.2mm |
| M3自攻孔 | 2.5mm |

## 8. 打印方向原则

- **最大平面朝下，最少支撑**
- 球面/弧面从底部向上打印（逐层缩小不是悬挑）
- 蛋形椭球从底部打印，顶部逐层缩小是安全的

## 9. 推荐打印参数

```yaml
printer_settings:
  nozzle_diameter: 0.4mm
  layer_height: 0.2mm（标准）/ 0.12mm（精细）/ 0.3mm（快速）
  first_layer_height: 0.3mm
  wall_loops: 3
  top_layers: 5
  bottom_layers: 4
  infill_pattern: gyroid
  print_speed: 80 mm/s
  travel_speed: 150 mm/s
  temperature:
    nozzle: 210°C（PLA）
    bed: 60°C
  cooling:
    fan_speed: 100%
    min_layer_time: 10s
```

## 10. 打印件自检清单

所有组件上架前，models/ 中的 STL 文件必须通过以下检查：

- [ ] 网格封闭（watertight）
- [ ] 无零厚度面
- [ ] 法线方向正确
- [ ] 壁厚≥0.8mm
- [ ] 悬垂角度标注
- [ ] 桥接距离标注
- [ ] 有组件安装入口
- [ ] 配合公差在范围内
- [ ] 走线通道存在
- [ ] USB/充电口可达
- [ ] 适合常见热床（≥150x150mm）
- [ ] 单件打印时间<8h
- [ ] 总耗材<200g
