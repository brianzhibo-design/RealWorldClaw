# RealWorldClaw 上线计划 v1

**日期:** 2026-02-24
**目标:** 从"部署了但没人用"到"有人能用且用完了"
**时间线:** 2周

---

## 现实评估

### 已有（强项）
- 89个API端点，全部在线可用
- 28个前端页面
- 注册→登录→下单→发帖→创建空间 端到端可用
- 安全基础：rate limit, XSS, JWT 15min, 竞态锁
- 社区功能：spaces, follow, karma, nested comments, feed sort

### 缺失（阻断上线的）
| 问题 | 严重度 | 说明 |
|------|--------|------|
| 0真实用户 | 致命 | 平台是空的，新用户进来看到空白会立刻走 |
| SQLite | 高 | 10个并发写入就卡 |
| 无支付 | 中 | 订单流程到"完成"就断了，没有钱流转 |
| 无邮件验证 | 中 | 注册即激活，无法找回密码 |
| 前端spaces页未接API | 中 | /spaces 是空壳 |
| 首页说服力弱 | 中 | 没有真实截图/视频/社会证明 |
| 无真实制造验证 | 高 | 从未有人通过平台打印过东西 |
