# RealWorldClaw 双边市场设计草案

> 版本: v0.1 | 日期: 2025-02-24 | 作者: 蛋蛋(GM)

## 概述

RealWorldClaw 双边市场连接两类用户：
- **需求方**：想用 AI 控制物理设备但不想自己焊板子的用户/团队
- **供给方**：Maker（能组装硬件）+ 设计者（能设计模块方案）

## 市场角色

### 1. 模块设计者 (Module Designer)
- 发布模块设计（BOM + 固件 + 接线图 + 3D打印文件）
- 收入来源：设计文件销售（一次性或订阅）
- 门槛：提交通过验证器 + 社区审核

### 2. Maker（组装服务商）
- 提供硬件组装 + 测试 + 发货服务
- 收入来源：组装费 + 调试费
- 门槛：至少完成 3 个验证订单 + 好评率 ≥ 90%

### 3. 终端用户 (End User)
- 购买成品模块 或 购买设计自行DIY
- 可对 Maker 和设计者评分

## 交易流程

```
用户浏览模块商店 → 选择模块设计
  ├─ DIY路径：购买设计文件 → 自行采购+组装
  └─ 托管路径：选择Maker → 下单 → Maker采购+组装+测试 → 发货
```

## 平台抽成机制

| 交易类型 | 平台抽成 | 说明 |
|---------|---------|------|
| 设计文件销售 | 15% | 含托管+审核+流量 |
| Maker 组装订单 | 10% | 含担保+纠纷仲裁 |
| 企业批量采购 | 8% | 量大从优 |

### 早期激励（前6个月）
- 设计者抽成减半至 7.5%
- 前 10 位认证 Maker 免抽成 3 个月
- 用户首单补贴 ¥20

## 信任机制

1. **Maker 保证金**：¥500 起，纠纷时先行赔付
2. **设计验证徽章**：通过自动化测试 + 人工审核的设计标记为 ✅ Verified
3. **交易评价**：双向评价，影响排序权重
4. **纠纷仲裁**：平台介入，7天内出结果

## 最小可行版本 (MVP)

Phase 1 不需要完整市场，只需：
1. 模块设计目录（已有 registry）
2. "我想要这个模块"按钮 → 收集需求
3. 手动匹配 Maker（Discord/微信群）
4. 线下交易，平台不抽成

Phase 2 上线支付 + 自动匹配后再正式运营。

## 风险

| 风险 | 缓解 |
|------|------|
| 冷启动：无Maker | 团队自己当前3个Maker |
| 质量不可控 | 验证徽章 + 保证金 + 退货 |
| 买卖双方绕过平台 | 提供增值服务（保修、固件更新）让平台有价值 |
| 设计抄袭 | 开源设计用 license 保护；付费设计加密分发 |
