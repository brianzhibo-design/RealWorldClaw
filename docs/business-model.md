# RealWorldClaw Business Model (Draft)

> Status: Draft v0.1 | 2026-02-22

## Model: Open Core + Marketplace

### Open Source (Apache 2.0)
- Energy Core firmware
- Module protocol specification
- 3D print files (STL/STEP)
- Basic SDK and CLI tools
- Reference module designs

### Commercial Layer

#### 1. Maker Network Commission
- Standard orders: **15%** platform fee
- Rush orders: **20%** platform fee
- Privacy: buyer/seller communicate through platform only

#### 2. Module Marketplace
- Free to list open-source modules
- Premium listings with verified badge: **$5/month**
- Commercial module sales: **10%** transaction fee

#### 3. SaaS Tiers

| Tier | Price | Features |
|------|-------|----------|
| **Free** | $0 | 3 agents, 1 device, community support |
| **Pro** | $19/mo | Unlimited agents, 10 devices, cloud logging, priority support |
| **Enterprise** | Custom | Multi-org, audit export, SLA, on-prem option |

#### 4. Hardware (Future)
- Pre-assembled Energy Core kits
- Certified module bundles
- Enterprise hardware packages

### Revenue Projections (Year 1 Target)
- Focus on community growth, not revenue
- Target: 1,000 registered AI agents, 100 active makers
- Revenue: Maker commissions + early Pro subscriptions

### Competitive Moat
- **Network effects**: More modules → more useful → more agents → more makers
- **Protocol lock-in**: RWC Bus standard becomes ecosystem standard
- **Community**: Open source trust + maker culture
- **AI-native**: Built for AI agents, not human-first IoT retrofit
