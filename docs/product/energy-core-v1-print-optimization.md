# Energy Core V1 — 3D打印工程优化报告
*By 小灰灰🐺 | 羊村公司技术部硬件工程师 | 2026-02-22*

## 📋 产品概要回顾
- **外壳**：100×100×100mm 正方体，分前壳(15mm深)＋后壳(85mm深)
- **内装**：ESP32-S3一体板 (50×86×9.1mm)，2.8寸屏幕，∅28mm喇叭
- **材质**：PLA/PETG，壁厚2.5mm
- **目标打印机**：拓竹 Bambu Lab P2S (256³mm打印体积，0.4mm喷嘴)

---

## 🎯 1. 打印方向与支撑分析

### 1.1 前壳最佳打印方向
**推荐方向**：屏幕面朝下（-Y方向朝向热床）

**分析**：
- **优势**：
  - 屏幕窗口完全无需支撑（成为底面开口）
  - 顶部十字槽朝上，易于后处理
  - 外观面（屏幕边框）完全无层纹
- **支撑需求**：
  - 十字槽：深度2mm，桥接长度24mm，需要支撑
  - 内部安装孔：深度15mm，需要支撑
- **悬空角度**：所有外表面均≤45°，结构合理

### 1.2 后壳最佳打印方向
**推荐方向**：Type-C开口面朝下（-Y方向朝向热床）

**分析**：
- **优势**：
  - Type-C开口无需支撑（底面开口）
  - 喇叭格栅孔朝上，易于打印
  - 安装柱朝上，无需支撑
- **支撑需求**：
  - 喇叭格栅：6×6阵列∅3mm孔，深度2.5mm，**建议取消支撑**（孔径够大）
  - 内部加强筋：需要**树状支撑**，消耗材料较多
- **悬空角度**：后壳深度85mm，打印时间约6-7小时

### 1.3 支撑优化策略
1. **十字槽**：使用**锥形支撑**，密度15%，易于拆除
2. **加强筋**：改为**从底部生长**的一体结构，避免支撑
3. **小孔**：≥∅3mm的孔无需支撑（Bambu P2S 0.4mm喷嘴可桥接）

## 🔧 2. 结构优化

### 2.1 壁厚分析与调整

**现状**：统一2.5mm壁厚

**优化建议**：
| 区域 | 现有厚度 | 建议厚度 | 原因 |
|------|----------|----------|------|
| 前壳外壁 | 2.5mm | **3.0mm** | 屏幕窗口周围需要更高强度 |
| 后壳外壁 | 2.5mm | **2.5mm** | 保持，符合FDM最小厚度 |
| 安装柱壁 | 1.5mm | **2.0mm** | 避免螺丝拧入时开裂 |
| 顶部槽壁 | 2.5mm | **1.8mm** | 减重，保持配合精度 |
| 底部脚垫 | 2.5mm | **4.0mm** | 增强接触面稳定性 |

### 2.2 前后壳对接方式

**现状分析**：脚本中未明确定义对接方式

**推荐方案**：**磁吸卡扣组合**
```
前壳边缘设计：
├─ 外圈：卡扣凸起 (4个角+4个边中点，共8个)
├─ 内圈：磁铁槽 (∅6×2mm，嵌入强磁)
└─ 密封：硅胶垫槽 (1mm深×3mm宽)

后壳边缘设计：
├─ 外圈：卡扣凹槽 (对应8个位置)
├─ 内圈：铁片槽 (0.5mm铁片，与磁铁吸附)
└─ 密封：硅胶垫对接面
```

**具体参数**：
- **卡扣尺寸**：长8mm，宽3mm，凸起0.8mm
- **倾斜角**：25°（易于拆装）
- **过盈配合**：0.1mm（PLA收缩补偿后）
- **磁铁规格**：N35 ∅6×2mm，8个位置

### 2.3 安装柱优化设计

**现状**：∅3.2mm螺纹孔，外径6.2mm

**优化方案**：
```
安装柱设计：
├─ 外径：8.0mm (增强强度)
├─ 内径：2.6mm (配M3自攻螺丝)
├─ 高度：12.0mm (PCB厚9.1mm + 3mm缓冲)
├─ 壁厚：2.7mm (外径-内径)/2
├─ 顶部倒角：0.5mm×45° (导向作用)
└─ 底部圆角：R1.0mm (减少应力集中)
```

**热熔螺母方案**：
- 规格：M3×4.5mm 热熔螺母
- 预留槽：∅5.4mm×4.5mm深
- 加热温度：180°C (PLA) / 200°C (PETG)

### 2.4 加强筋优化设计

**现状问题**：3条加强筋需要大量支撑

**优化方案**：**蜂窝加强结构**
```
新加强筋布局：
├─ 底部蜂窝：六边形镂空，壁厚1.2mm
├─ 侧壁加强：垂直筋条，厚度1.0mm，间距20mm
├─ PCB支撑台：4个对角支撑，高度可调
└─ 喇叭固定环：∅28mm内环，集成在后壁
```

**具体参数**：
- **蜂窝边长**：12mm
- **筋条高度**：30mm（不超过PCB位置）
- **材料节省**：相比实心加强筋节省40%材料
- **打印时间**：减少1.5小时

### 2.5 顶部十字槽优化

**现状**：24mm臂长，3mm宽，2mm深

**优化建议**：
| 参数 | 现有值 | 优化值 | 说明 |
|------|--------|--------|------|
| 臂长 | 24mm | **22mm** | 避免槽端应力集中 |
| 槽宽 | 3mm | **3.2mm** | 配合公差+0.2mm |
| 槽深 | 2mm | **2.5mm** | 增强配合稳定性 |
| 端部圆角 | 无 | **R1.5mm** | 美观且易于脱模 |
| 底部倒角 | 无 | **0.3mm×45°** | 手办插入导向 |

## 📏 3. 公差与配合

### 3.1 外壳内径 vs PCB外径间隙

**PCB实际尺寸**：50×86×9.1mm
**推荐间隙**：
- **X方向**：±1.0mm（总间隙2.0mm）
- **Z方向**：±0.8mm（总间隙1.6mm）  
- **Y方向**：±2.0mm（安装/维修空间）

**计算依据**：
- PLA打印精度：±0.15mm
- 装配公差：±0.3mm  
- 热胀冷缩：±0.1mm
- 安全余量：0.45mm

### 3.2 屏幕窗口公差

**屏幕显示区**：43.2×57.6mm
**窗口开孔尺寸**：44.0×58.4mm
**公差设计**：
- **显示区余量**：上下左右各+0.4mm
- **倒角设计**：内侧45°×0.5mm（减少锐边）
- **深度公差**：窗口深度2.5mm±0.1mm

### 3.3 Type-C开口公差

**Type-C尺寸**：9.0×3.2mm
**开孔建议**：10.0×4.0mm
**公差分析**：
- **宽度余量**：+1.0mm（插拔空间）
- **高度余量**：+0.8mm（上下对齐误差）
- **深度**：穿透后壁+1mm（避免干涉）

### 3.4 前后壳配合公差

**配合类型**：过渡配合（H7/h6级别）

| 特征 | 公差带 | 数值 |
|------|--------|------|
| 前壳凸台外径 | h6 | 99.8±0.1mm |
| 后壳凹槽内径 | H7 | 100.0±0.15mm |
| **配合间隙** | - | **0.05~0.35mm** |
| 密封槽公差 | - | 3.0±0.1mm |

### 3.5 3D打印收缩补偿

**Bambu P2S + PLA收缩率**：0.15%

**补偿策略**：
```python
# 在Blender脚本中应用收缩补偿
SHRINKAGE_FACTOR = 1.0015  # 0.15%收缩率补偿

# 关键尺寸补偿
SCREW_HOLE_DIA = 2.6 / SHRINKAGE_FACTOR  # M3自攻孔
TYPEC_OPENING_W = 10.0 / SHRINKAGE_FACTOR  # Type-C开口
SCREEN_CUTOUT_W = 44.0 / SHRINKAGE_FACTOR  # 屏幕窗口
```

## 🌡️ 4. 散热设计

### 4.1 ESP32-S3功耗热量评估

**工作模式分析**：
| 模式 | CPU负载 | WiFi状态 | 屏幕状态 | 功耗 | 发热量 |
|------|---------|----------|----------|------|--------|
| 待机 | 1% | 间歇连接 | 息屏 | 0.05W | 可忽略 |
| 正常 | 30% | 持续连接 | 50%亮度 | 1.2W | **1.0W** |
| 峰值 | 100% | 高速传输 | 100%亮度 | 2.5W | **2.0W** |

**热阻分析**：
- PCB到空气：15°C/W
- 外壳内部对流：25°C/W  
- **总热阻**：40°C/W
- **温升**：正常80°C，峰值120°C ⚠️

### 4.2 通风口设计方案

**必要性**：峰值工作时可能过热，需要被动散热

**通风口布局**：
```
进风口（后壳底部）：
├─ 位置：Type-C开口下方20mm
├─ 形式：百叶窗式，6条×15mm长
├─ 倾斜角：45°向上（防尘防水）
└─ 开口率：30%（平衡散热与强度）

出风口（前壳顶部）：
├─ 位置：十字槽两侧
├─ 形式：隐藏式格栅，融入外观
├─ 尺寸：2×(20mm×2mm)细缝
└─ 位置：不影响手办安装
```

**散热效果预测**：
- 自然对流：温升降低30%
- 工作温度：正常56°C，峰值84°C ✅
- 外壳表面：最高55°C（安全）

### 4.3 导热优化

**金属导热片方案**（选配）：
- 材质：0.5mm厚铜片
- 尺寸：20×20mm
- 位置：ESP32芯片背面
- 固定：导热胶贴
- 效果：温升再降15%

## 🔧 5. 可维护性

### 5.1 PCB安装/拆卸方案

**当前问题**：PCB被前壳屏幕窗口限制，拆卸困难

**优化方案**：
```
安装步骤：
1. 后壳放置，安装柱朝上
2. PCB从前方插入，屏幕先进
3. PCB滑入安装柱导向槽
4. M3×8mm自攻螺丝固定（4颗）
5. 前壳对接，卡扣锁定

拆卸步骤：
1. 掰开8个卡扣（无工具）
2. 分离前后壳
3. 拧出4颗螺丝
4. PCB从前方抽出
```

**关键改进**：
- 安装柱顶部设**导向倒角**
- PCB底部开**定位缺口**（2×2mm）
- 螺丝孔位设**漏斗形导入**（∅4mm→∅2.6mm）

### 5.2 喇叭固定方案

**当前问题**：喇叭悬空，依赖连接线支撑

**优化方案**：
```
喇叭安装架（后壳一体）：
├─ 圆形固定环：∅30mm，容纳∅28mm喇叭
├─ 卡扣固定：3个120°均布的弹性卡扣
├─ 导线槽：3mm宽×2mm深，引导到PCB
└─ 减振垫：硅胶垫圈，厚度1mm
```

**安装步骤**：
1. 喇叭插入固定环
2. 3个卡扣自动锁定
3. 导线从槽内走线到PCB

### 5.3 无工具拆装设计

**卡扣优化**：
- **手指操作区**：卡扣旁开设3mm深握槽
- **视觉标识**：卡扣位置丝印小箭头
- **触感反馈**：卡扣按压时有明显"咔哒"声
- **颜色区分**：卡扣区域使用对比色AMS材料

**磁吸强度**：
- 单个磁铁吸力：2N
- 8个磁铁总吸力：16N  
- 手动分离力：约5N（舒适范围）

## 🎯 6. 打印参数建议

### 6.1 Bambu P2S最优打印设置

**基础参数**：
| 参数 | 前壳设置 | 后壳设置 | 说明 |
|------|----------|----------|------|
| **层高** | 0.15mm | 0.20mm | 前壳需要更好表面质量 |
| **填充率** | 20% | 15% | 前壳承受更多应力 |
| **外壳层数** | 4层 | 3层 | 前壳1.2mm，后壳0.9mm |
| **打印速度** | 80mm/s | 100mm/s | 前壳细节多，需要慢速 |
| **喷嘴温度** | 215°C | 210°C | PLA+，前壳需要更好层间结合 |

### 6.2 关键特征特殊处理

**薄壁特征**（<2mm）：
```
参数调整：
├─ 层高：0.12mm（提高精度）
├─ 线宽：0.35mm（略小于喷嘴）
├─ 流量：95%（避免过挤）
└─ 速度：60mm/s（保证质量）
```

**孔洞特征**（∅2-5mm）：
```
参数调整：
├─ 桥接速度：30mm/s
├─ 桥接流量：90%
├─ 冷却风扇：100%
└─ 回抽设置：1.2mm×25mm/s
```

**卡扣特征**：
```
关键设置：
├─ 层高：0.10mm（关键配合面）
├─ 外壁速度：40mm/s
├─ 线性提前：0.02mm（锐角补偿）
└─ 后处理：600目砂纸轻磨
```

### 6.3 AMS多色方案建议

**方案A：功能分色**
- **前壳**：象牙白（主体）+ 天空蓝（屏幕边框）
- **后壳**：深灰色（主体）+ 橙色（接口标识）
- **优势**：功能区分明确，维修识别方便

**方案B：美观配色**
- **前壳**：珍珠白（单色，高端感）
- **后壳**：太空灰（单色，商务感）
- **优势**：外观统一，成本较低

**推荐**：方案A（符合羊村风格🐺）

### 6.4 后处理建议

2. **打磨**：400目→800目→1500目砂纸
3. **抛光**：细砂布+抛光膏（关键配合面）
