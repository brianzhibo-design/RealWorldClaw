## 2. 磁铁选型

### 2.1 钕磁铁等级对比

| 等级 | 最大磁能积 (MGOe) | 剩磁 (T) | 矫顽力 | 最高工作温度 | 相对价格 |

| N35 | 33-36 | 1.17-1.21 | 标准 | 80°C | 1x |
| N42 | 40-43 | 1.28-1.32 | 中等 | 80°C | 1.3x |
| N52 | 48-51 | 1.42-1.47 | 较高 | 60°C⚠️ | 2x |

### 2.2 5×2mm圆柱磁铁

对于5mm直径×2mm高的圆柱钕磁铁：
- **N35**：吸力约 0.4-0.5 kg — 偏弱，不推荐
- **N42**：吸力约 0.6-0.8 kg — **推荐**，性价比最佳
- **N52**：吸力约 0.9-1.1 kg — 过强，可能难以分离；且最高工作温度低

**推荐：N42等级**
- 磁力适中，手指可轻松分离
- 工作温度80°C，安全余量充分
- 价格合理

### 2.3 防反插设计：对角极性排列

```
连接器布局（俯视）：

    [磁铁A: N极朝上]  [8个Pogo Pin]  [磁铁B: S极朝上]
    
    对配端：
    [磁铁A': S极朝上]  [8个Pogo Pin]  [磁铁B': N极朝上]
```

- 两个磁铁极性相反 → 正确方向时两对都吸合
- 翻转180°时 → 两对都排斥 → 无法插入
- 这是MagSafe和Surface Connect的核心防呆原理

### 3.3 3D打印外壳安装方式

推荐设计方案：

```
截面图：

    ┌─────────────────────┐
    │  3D打印外壳          │
    │  ┌───┐ ┌─────┐ ┌───┐│
    │  │磁铁│ │PCB板 │ │磁铁││  ← 磁铁压入/胶粘孔位
    │  └───┘ │pogo │ └───┘│
    │        │pins │      │
    │        └──┬──┘      │
    │           │         │
    └───────────┴─────────┘
                ↓ pin露出表面
```

**安装要点**：
1. **磁铁孔位**：设计Ø5.1mm × 2.1mm的盲孔（留0.1mm间隙），磁铁用502胶固定
2. **PCB固定**：M2螺丝或卡扣式，PCB上焊接pogo pin
3. **对齐导向**：外壳边缘加0.5mm凸台/凹槽，辅助对位
4. **公差**：FDM打印建议预留0.2mm间隙，SLA可用0.1mm

## 5. RWC Bus适配建议

基于RWC Bus的8pin定义（I2C SDA/SCL + 5V/GND + 4扩展pin）：

1. **先用方案A验证**：买现成8pin磁吸连接器，验证I2C信号通过pogo pin的可靠性
2. **关注接触电阻**：I2C对接触电阻不敏感（数字信号），但建议测量验证
3. **电流预算**：5V供电如果走单pin 2A，总功率10W，绰绰有余
4. **后续如需定制**：转方案B，PCB上可以优化pin排列（电源pin在两端，信号pin在中间，减少串扰）

### Pin排列建议（自定义时）

```
[GND] [SCL] [SDA] [AUX1] [AUX2] [AUX3] [AUX4] [5V]
  ↑                                               ↑
 电源pin在两端，信号pin在中间
```
