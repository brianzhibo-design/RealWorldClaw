## 1. FDM打印技术核心

### 关键参数及影响

| 参数 | 范围 | 对质量的影响 | 对时间的影响 |
|------|------|-------------|-------------|
| **层高** | 0.08-0.32mm | 越低越精细，0.2mm是黄金平衡点 | 0.12mm比0.28mm慢2倍+ |
| **填充率** | 0-100% | 15-20%足够日常，40%+结构件 | 每+20%填充约+15%时间 |
| **壁厚** | 2-4圈（0.8-1.6mm） | 3圈(1.2mm)是产品级最低标准 | 影响小 |
| **打印速度** | 50-600mm/s | 越快表面越粗糙，P2S 300mm/s质量仍优 | 线性关系 |
| **喷嘴温度** | 190-260°C | 太低层间粘合差，太高拉丝 | 无直接影响 |
| **热床温度** | 50-110°C | 太低翘曲，太高象脚 | 无直接影响 |

### 支撑结构
- **45°规则**：悬挑角度>45°需要支撑
- **树状支撑**（Bambu默认）：省料、易拆除、痕迹小
- **设计时优先避免支撑**：旋转模型、加倒角、分件打印

### 常见缺陷排查

| 缺陷 | 原因 | 解决 |
|------|------|------|
| **翘曲** | 热床温度低/无胶/环境温差 | 加热床温度、涂胶棒、封箱体 |
| **拉丝** | 回抽不足/温度过高 | 增加回抽距离(1-2mm)、降温5°C |
| **层裂/分层** | 温度低/层间粘合差 | 升温、降速、减小层高 |
| **象脚** | 首层压得太紧/热床太热 | 调Z偏移、降热床温度5°C |
| **堵头** | 热端积碳/耗材受潮 | 冷拉清洁、烘干耗材 |
| **表面粗糙** | 速度太快/振动 | 降速、检查皮带张力 |

## 2. 材料学

### 主流材料对比

| 材料 | 强度 | 耐温 | 柔性 | 打印难度 | 价格/kg | 最佳场景 |
|------|------|------|------|---------|---------|---------|
| **PLA** | ★★★ | 56°C | 脆 | ★☆☆ | ¥60-80 | 外壳原型、展示件 |
| **PETG** | ★★★★ | 70°C | 微弹 | ★★☆ | ¥70-100 | 功能件、耐用外壳 |
| **ABS** | ★★★★ | 98°C | 中 | ★★★ | ¥60-80 | 耐热结构件（需封箱） |
| **TPU** | ★★ | 60°C | 高弹 | ★★★ | ¥100-150 | 减震垫、柔性连接 |
| **ASA** | ★★★★ | 95°C | 中 | ★★★ | ¥80-120 | 户外件（抗UV） |
| **Nylon/PA** | ★★★★★ | 80°C | 韧 | ★★★★ | ¥150-300 | 齿轮、铰链、高强度件 |
| **PC** | ★★★★★ | 130°C | 韧 | ★★★★★ | ¥200-400 | 透明件、极端耐热 |

### Energy Core材料选择决策
- **外壳**：PLA（展示/原型）→ PETG（量产，更耐用）
- **电池仓**：PETG（微弹性便于电池安装）
- **卡扣/连接件**：PETG或Nylon（需要韧性，不能脆断）

### 材料存储
- PLA/PETG：密封袋+干燥剂，湿度<20%
- Nylon/TPU：必须烘干后使用（Bambu AMS自带烘干功能）
- 受潮症状：打印时噼啪声、表面气泡、层间粘合差

## 3. DfAM — 面向3D打印的设计

### 核心设计规范（FDM / P2S）

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| **最小壁厚** | 1.2mm (3圈@0.4喷嘴) | 低于0.8mm结构不可靠 |
| **最小特征尺寸** | 0.5mm | 低于此无法可靠打印 |
| **装配间隙** | 0.3-0.5mm（单边） | FDM精度约±0.2mm |
| **卡扣间隙** | 0.5mm | 配合倒角/锥面引导 |
| **最大悬挑** | 45° | 超过需支撑 |
| **孔洞最小直径** | 2mm | 更小的孔需后处理钻扩 |
| **螺纹** | M3以上 | M3以下用热熔嵌件 |
| **倒角/圆角** | 所有直角加R0.5+ | 减少应力集中+避免翘曲 |

### 关键设计原则

**打印方向决定强度**
- 层间（Z方向）强度仅为层内的30-50%
- 受力方向应平行于打印层
- Energy Core外壳：竖直打印，受力在水平面

**减少支撑的技巧**
- 悬挑<45°无需支撑
- 用倒角替代直角悬空
- 桥接（两端有支撑的水平面）可达50mm
- 水滴形孔洞替代圆形孔洞

**多件拼装 vs 一体打印**
- 超过200mm或有内部结构：分件
- 需要不同材料：分件
- 卡扣/螺丝连接设计要预留间隙
- Energy Core方案：5件拼装（top-shell, bottom-shell, screen-frame, battery-tray, pcb-standoff）— 正确决策

## 4. 后处理工艺

| 工艺 | 适用 | 效果 | 成本 |
|------|------|------|------|
| **砂纸打磨** | 所有材料 | 消除层纹（120→400→800目） | 低 |
| **底漆+喷漆** | PLA/ABS | 隐藏层纹，产品级外观 | 中 |
| **热熔嵌件** | 螺纹安装 | M2-M5黄铜嵌件，热压入 | ¥0.1/个 |
| **胶合** | 多件拼装 | 502瞬干胶（PLA）/ AB胶（PETG） | 低 |
| **丙酮蒸汽** | 仅ABS/ASA | 表面光滑如注塑 | 低但需通风 |
| **填缝打磨** | 拼接缝 | 原子灰填缝→打磨→喷漆 | 中 |

### Energy Core后处理流程
1. 拆除支撑（如有）
2. 砂纸打磨拼接面（确保配合）
3. 装入电子件
4. 卡扣/螺丝组装
5. （可选）喷漆上色

## 6. 产品级3D打印

### 成本核算公式

```
单件成本 = 材料成本 + 电费 + 时间成本 + 设备折旧 + 失败率摊销

材料成本 = 耗材重量(g) × 单价(¥/g)
  PLA: ~¥0.07/g, PETG: ~¥0.09/g

电费 = 功率(kW) × 时间(h) × 电价(¥/kWh)
  P2S平均功耗: ~150W, 电价~¥0.6/kWh
  1小时电费 ≈ ¥0.09

时间成本 = 打印时间(h) × 时薪(如适用)

设备折旧 = 设备价格 / 预计使用小时数
  P2S ~¥4000 / 5000h = ¥0.8/h

失败率摊销: 初期5-10%, 成熟后<2%
```

### Energy Core成本估算
- 外壳5件总重约40g PLA → 材料¥2.8
- 打印时间约2小时 → 电费¥0.18 + 折旧¥1.6
- **单套打印成本 ≈ ¥5**（不含电子件）

### 何时转注塑？
- **<500件/月**：3D打印更经济（无模具费）
- **500-2000件/月**：混合方案（标准件注塑+定制件打印）
- **>2000件/月**：注塑（模具¥5000-50000，但单件<¥1）
- **Energy Core策略**：前期全3D打印验证，月销>1000后考虑注塑外壳

### 质量一致性控制
- 首件检验：每批次首件量尺寸
- 关键尺寸公差：±0.2mm（P2S可达）
- 称重验证：同型号重量偏差<5%
- 定期校准：每100小时校准热床和XY
- 环境控制：温度20-25°C，湿度<60%

## 7. 行业趋势（2025-2026）

### 已成熟
- **多色打印**：AMS系统4色已标配，16色可扩展
- **AI缺陷检测**：P2S的6种AI检测已相当可靠
- **高速打印**：600mm/s级别已普及

### 正在发展
- **多材料打印**：同一打印件混合刚性+柔性材料
- **连续碳纤维**：Markforged路线，强度接近铝合金
- **桌面级金属打印**：Desktop Metal等，价格仍>$50K

### 对RealWorldClaw的影响
- 多色打印=模块外壳可更美观（无需喷漆）
- AI检测=Maker Network质量保证的技术基础
- 高速打印=订单交付时间缩短，平台竞争力
