### 专业建模标准总结
- **面数**: 低多边形版本 5K-50K面（实时渲染）；高多边形 100K-500K面（产品渲染）
- **细分度**: SubD建模通常2-3级细分
- **材质**: PBR工作流（Base Color、Metallic、Roughness、Normal Map）
  - 塑料: Roughness 0.3-0.6, Metallic 0
  - 金属: Roughness 0.1-0.3, Metallic 1.0
  - 橡胶: Roughness 0.7-0.9, Metallic 0
- **渲染**: HDRI环境光 + 三点布光，地面反射

## 三、消费电子3D外壳设计趋势总结

### 1. 组装方式趋势
- **卡扣式（Snap-fit）成为主流** — 无螺丝设计降低组装门槛
- **热熔螺母（Heat-set inserts）** 用于需要反复拆装的场景
- **摩擦配合（Friction fit）** 用于显示屏等精密部件
- 螺丝仅在需要IP等级防护时使用

### 2. 外观设计趋势
- **圆润边角** — 绝大多数热门模型采用倒角/圆角处理
- **极简主义** — 表面干净，功能性开孔（USB、散热）精确且对称
- **双色/多色打印** — 拓竹AMS多色能力被广泛利用
- **纹理填充** — 利用切片器的填充图案作为装饰元素

### 3. 工程参数标准
| 参数 | 推荐值 |
|------|--------|
| 壁厚 | 2.0-2.5mm |
| PCB间隙 | ≥0.5mm每边 |
| 卡扣公差 | 0.2-0.3mm |
| 前面板边框 | 3-4mm |
| 层高 | 0.2mm（功能件）/ 0.12mm（外观件） |
| 填充 | 10-20%（外壳类） |
| 支撑 | 尽量避免（设计时考虑打印方向） |

### 4. 磁吸连接趋势
- 6x3mm 圆形钕磁铁为最常用规格
- 磁铁通过压入孔固定（孔径比磁铁大0.1-0.2mm）
- Pogo pin通常4-5pin，间距2.54mm或自定义
- 磁吸+pogo pin组合已在DIY社区得到验证

### 5. 散热设计
- 蜂窝状散热孔
- WiFi/功能性图标散热孔（一举两得）
- ESP32发热量需要考虑通风

## 四、针对RealWorldClaw Pulse的设计建议

### 核心设计方向
1. **圆形/圆柱形主体** — 匹配1.28寸圆形显示屏的圆形语言
2. **卡扣+磁吸双方案**
   - 主体：卡扣式上下壳（无螺丝）
   - 充电底座：磁吸对准 + Pogo Pin充电/数据传输
3. **前面板设计**
   - 圆形显示窗口，边框3-4mm
   - 摩擦配合嵌入显示屏
   - 可考虑透明/半透明前面板环（呼吸灯效果）
4. **壁厚2mm，PCB间隙0.5mm**
5. **底部设计**
   - 磁铁嵌入位（6x3mm钕磁铁 x4）
   - Pogo Pin触点窗口
   - 橡胶脚垫凹槽

### 材质建议
- **外壳**: PLA/PETG（原型期），后期考虑PC或注塑
- **布艺/硅胶外套**: 参考Home Assistant Voice PE设计，提升消费品质感
- **底座**: 加重底座（配重块），增加稳定性

### 需要注意的设计细节
- ESP32天线区域不能有金属/密闭遮挡
- USB-C口尽量设计在底部或背部，正面保持干净
- 按钮（如果有）使用硅胶帽或薄壁传力设计
- 考虑LED指示灯的导光柱/透光窗
