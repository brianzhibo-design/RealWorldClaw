## 1. 打印质量检测方法

### 1.1 尺寸精度检测

**工具：** 数显卡尺（精度0.01mm）、钢尺、塞规

**测量要点：**
- X/Y方向（水平）精度通常优于Z方向（层高方向）
- FDM打印典型精度：±0.2mm（P2S可达±0.1mm）
- 孔径通常偏小（热收缩+挤出膨胀），轴径通常偏大

**关键测量位：**
| 特征类型 | 测量方法 | 注意事项 |
|---------|---------|---------|
| 外轮廓尺寸 | 卡尺外测量 | 取3个位置平均值 |
| 孔径 | 卡尺内测量/塞规 | 在孔口和孔中部各测一次 |
| 壁厚 | 卡尺测量 | 至少测3处 |
| 高度 | 卡尺深度测量 | 放平面上测量 |
| 配合面间距 | 卡尺/塞规 | 反复测量确认一致性 |

**公差记录表模板：**

| 特征 | 设计值(mm) | 实测值(mm) | 偏差(mm) | 合格(±0.2mm) |
|------|-----------|-----------|---------|-------------|
| 外壳长度 | 100.0 | | | ☐ |
| 外壳宽度 | 60.0 | | | ☐ |
| USB-C开口宽 | 9.2 | | | ☐ |
| 卡扣孔径 | 3.0 | | | ☐ |

### 1.2 表面质量评估

| 缺陷类型 | 表现 | 评级标准 |
|---------|------|---------|
| **层纹** | 水平线条可见 | A:不可见 B:可见但光滑 C:可触摸到 D:严重台阶 |
| **拉丝** | 细丝连接于不同部位之间 | A:无 B:极少可擦除 C:需修剪 D:严重影响外观 |
| **翘边** | 底部边缘上翘脱离平面 | A:无 B:<0.5mm C:0.5-2mm D:>2mm |
| **象脚** | 底部第一层向外膨胀 | A:无 B:<0.3mm C:0.3-1mm D:>1mm |
| **顶面质量** | 顶部表面平整度 | A:光滑 B:轻微不平 C:有缝隙 D:塌陷 |
| **过挤出/欠挤出** | 表面凸起或凹陷 | 目测+触摸检查 |

**评估方法：**
1. 自然光下目视检查所有表面
2. 手指触摸检查光滑度
3. 与参考件对比（如有）
4. 拍照记录（固定角度+光源方便对比）

### 1.3 结构强度测试

**卡扣耐久测试：**
- 将卡扣反复插拔，记录次数
- 合格标准：≥50次插拔后仍能正常卡合
- 每10次检查一次：卡合力是否明显下降、有无白化/裂纹

**承重测试：**
- 在关键支撑面放置已知重量
- 记录：变形量、破裂时最大载荷
- 注意方向：层间方向（Z）强度最弱

**层间附着力简易测试：**
- 用手尝试沿层线方向掰开
- 不应轻易分层
- 若徒手可分层 → 打印温度或冷却设置有问题

### 1.4 配合精度测试

**松紧度评估标准：**
| 等级 | 描述 | 适用场景 |
|------|------|---------|
| 过松 | 自由晃动，有明显间隙 | 不合格 |
| 松配合 | 可轻松手动插入，轻微间隙 | 需要经常拆装的部分 |
| 适中 | 需轻微用力插入，无明显间隙 | 一般装配 |
| 紧配合 | 需较大力插入，过盈配合 | 不需要拆卸的永久连接 |
| 过紧 | 无法插入或强行插入导致变形/破裂 | 不合格 |

**测试方法：**
1. 干配合测试（不加任何辅助）
2. 记录插入力感受（主观1-5分）
3. 组装后晃动检查
4. 如需要，用塞尺测量间隙

## 2. 常见打印问题排查

### 2.1 翘边/层裂 (Warping / Bed Adhesion Failure)

**表现：** 打印件底部边缘翘起脱离热床，严重时整个件脱落

**原因：**
- 热床温度不够或不均匀
- 第一层附着力不足（热床不平、Z偏移不对）
- 材料收缩率高（ABS>PETG>PLA）
- 件大且扁平，收缩力累积
- 环境温风/开门导致温差

**排查步骤：**
1. 检查热床温度设置是否正确（PLA:55-60°C, PETG:70-80°C, ABS:90-110°C）
2. 检查第一层Z偏移（挤压是否足够）
3. 检查热床是否清洁（酒精擦拭PEI板）
4. 检查环境是否有风吹

**解决方法：**
- 提高热床温度5-10°C
- 降低第一层速度（20-30mm/s）
- 增大第一层线宽（120%）
- 使用Brim（边缘裙边）增加附着面积
- ABS必须使用封闭箱体
- 清洁热床（IPA酒精）

### 2.2 拉丝/挂丝 (Stringing)

**表现：** 打印件不同部位之间有细丝连接

**原因：**
- 回抽距离不足或回抽速度不够
- 打印温度过高导致材料过稀
- 空走速度太慢
- 材料吸潮

**排查步骤：**
1. 打印拉丝测试塔（Stringing Test Tower）
2. 检查当前回抽设置
3. 检查打印温度是否偏高
4. 检查材料是否干燥

**解决方法：**
- 增加回抽距离（直驱:0.5-2mm, 远程:4-6mm；P2S直驱约0.8mm起调）
- 提高回抽速度（30-50mm/s）
- 降低打印温度5-10°C
- 提高空走速度（150mm/s+）
- 干燥材料（食品干燥箱50°C 4-6h）
- 开启"Wipe while retract"

### 2.3 层间附着力不足 (Poor Layer Adhesion)

**表现：** 打印件沿层线方向容易掰开、强度低

**原因：**
- 打印温度偏低，层间融合不充分
- 层高过大（相对喷嘴直径）
- 冷却风扇过大
- 打印速度过快
- 材料吸潮

**排查步骤：**
1. 打印薄壁测试件，手动测试层间粘合力
2. 检查温度是否在材料推荐范围中高段
3. 检查风扇设置

**解决方法：**
- 提高打印温度5-15°C
- 降低层高（不超过喷嘴直径的75%，0.4mm喷嘴用≤0.3mm层高）
- 降低冷却风扇（PLA:100%→80%, PETG:50-70%, ABS:0-30%）
- 适当降低打印速度
- 增加线宽（提高挤出量）

### 2.4 尺寸不准 (Dimensional Inaccuracy)

**表现：** 打印件比设计尺寸偏大或偏小

**原因 - 偏大：**
- 挤出倍率过高（Over-extrusion）
- 线宽设置过大
- 热膨胀

**原因 - 偏小：**
- 挤出倍率不足（Under-extrusion）
- 材料收缩（冷却后）
- 皮肤/壁厚重叠设置

**原因 - 孔偏小：**
- 热收缩+内壁挤出膨胀（FDM通缺陷）
- 通常需要补偿0.2-0.4mm

**排查步骤：**
1. 打印校准立方体（20x20x20mm），测量三轴
2. 打印孔测试板（各种直径孔），测量实际孔径
3. 检查挤出倍率（流量校准）

**解决方法：**
- 校准挤出流量（Flow Calibration）
- 在CAD中预补偿：孔径加大0.2-0.4mm，外径按需调整
- 使用切片器的"水平尺寸补偿"功能（Bambu Studio有此设置）
- X/Y补偿：Bambu Studio → 打印机设置 → 高级

### 2.5 支撑去除痕迹 (Support Removal Marks)

**表现：** 去除支撑后表面粗糙、有疙瘩或凹坑

**原因：**
- 支撑与模型之间间距太小
- 支撑类型不适合
- 支撑界面层密度过高

**排查步骤：**
1. 检查支撑Z间距设置（通常1-2层高）
2. 检查支撑界面密度
3. 评估是否可以改变打印方向避免支撑

**解决方法：**
- 增大支撑顶部Z间距（0.2-0.3mm）
- 降低支撑界面密度（50-75%）
- 使用树形支撑（接触面积小）
- 重新设计零件减少悬垂（DFM优化）
- 后处理：砂纸打磨（120→240→400目）
- 优先改变打印方向，将外观面朝上

## 5. 学习总结与后续行动

### 关键认知
1. FDM打印的精度极限约±0.1-0.2mm，设计时需预留补偿
2. Z方向（层间）始终是最弱方向，结构设计需考虑打印方向
3. 孔径普遍偏小是FDM通病，设计时加大0.2-0.4mm补偿
4. 材料干燥度对打印质量影响巨大，PETG/ABS/Nylon尤其敏感
5. 系统化记录每次打印参数是优化的基础——不记录=没发生
