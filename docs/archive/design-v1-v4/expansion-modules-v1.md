# RealWorldClaw 扩展模块设计方案 v1

> 设计：沸羊羊🐏（研究总监）| 2026-02-21
>
> 核心原则：模块本身无AI无屏幕，纯传感器/执行器扩展，通过 RWC Bus（8pin 磁吸 Pogo Pin）连接主机。外壳3D打印，电子元件开源BOM。

## 1. 🌿 GreenThumb Module / 园艺模块

**一句话：** AI学会了种花——能感知土壤状态并自动浇水。

**核心硬件：**
- 电容式土壤湿度传感器 × 1（Capacitive Soil Moisture Sensor v1.2，抗腐蚀优于电阻式）
- 蠕动水泵 12V DC（流量 ~100ml/min）
- MOSFET驱动板（AO3400，控制水泵开关）
- 硅胶软管 + 水箱接口

**AI怎么用它：**
- 每小时读取土壤湿度，结合天气预报（联网）判断是否需要浇水
- 学习不同植物的浇水曲线（用户告诉AI种的是什么），避免过浇
- 出门旅行时自动托管，每天语音汇报："你的薄荷今天喝了50ml水，状态很好"
- 检测到土壤持续干燥且水泵已工作→判断水箱空了→提醒用户加水

**BOM成本：** ¥25-35

**3D打印难度：** 中等（需要防水设计，水泵固定座，水管卡扣）

**目标用户：** 植物爱好者、经常出差的人、阳台/办公桌绿植养护

**与主机交互：**
- 插入时屏幕显示🌱动画，AI说："园艺模块已连接！告诉我你种的是什么植物？"
- 日常屏幕显示土壤湿度条 + 上次浇水时间 + 小植物生长动画

## 2. ⚖️ SmartScale Module / 智能称重模块

**一句话：** AI有了"手感"——能称量物体重量并理解食物营养。

**核心硬件：**
- 称重传感器（Load Cell）50kg 半桥式
- HX711 24bit ADC模块
- 称重平台铝片（或3D打印刚性平台）

**AI怎么用它：**
- 用户放上食材，AI报出重量并自动查询营养成分（"这个苹果182g，约95大卡"）
- 烘焙助手模式：AI按配方逐步指导，"现在加面粉，目标250g……再加12g……好的，刚好！"
- 记录每日饮食重量，结合用户健康目标给出建议
- 快递称重：帮估算邮费

**BOM成本：** ¥12-18

**3D打印难度：** 简单（平台+底座，结构简单）

**目标用户：** 健身控卡人群、烘焙爱好者、咖啡手冲玩家（精确称豆子+注水量）

**与主机交互：**
- 插入时屏幕显示⚖️图标，AI说："称重模块就绪，把东西放上来试试！"
- 实时显示重量数字（大字体），AI根据上下文自动切换模式（厨房秤/咖啡秤/快递秤）

## 3. 🌡️ EnvSense Module / 环境感知模块

**一句话：** AI获得了"体感"——能感知温度、湿度、空气质量和光照。

**核心硬件：**
- SHT40 温湿度传感器（I2C，±0.2°C精度）
- PMS5003 激光PM2.5传感器
- BH1750 光照强度传感器（I2C）

**AI怎么用它：**
- "现在开窗通风好吗？"→AI综合室内PM2.5、室外空气质量指数、温度差给出建议
- 检测到PM2.5飙升→"你是不是在炒菜？建议开抽油烟机"
- 夜间光照传感器检测到灯还亮→结合时间提醒："已经凌晨1点了，该睡了"
- 长期记录环境数据，发现规律："你家每天下午3点最闷，建议那时开窗"

**BOM成本：** ¥55-75（PMS5003占大头约¥40）

**3D打印难度：** 中等（需要通风格栅设计，PM2.5传感器进出风口）

**目标用户：** 有娃家庭、过敏体质、空气质量敏感人群、智能家居爱好者

**与主机交互：**
- 插入时屏幕显示温湿度+空气质量仪表盘，AI说："环境感知已上线，我现在能感受到这个房间了"
- 常驻显示：温度/湿度/PM2.5/光照 四格小组件

## 4. 📷 Vision Module / 视觉模块

**一句话：** AI有了"眼睛"——能看见并理解周围的世界。

**核心硬件：**
- OV5640 500万像素摄像头模块（自动对焦，USB或MIPI接口转SPI）
- 可选：补光LED × 2（白光）
- 广角镜头（120° FOV）

**AI怎么用它：**
- 对准冰箱："你冰箱里有鸡蛋、牛奶、西兰花，我建议今晚做个西兰花炒蛋"
- 对准书架：识别书脊，帮你建立书籍清单
- 拍文档/手写笔记→OCR提取文字→存档或发送
- 植物诊断：拍叶子判断是否有病虫害（结合园艺模块）
- 宠物监控：检测到猫跳上桌子→语音提醒

**BOM成本：** ¥25-40

**3D打印难度：** 简单（摄像头固定座+可调角度支架）

**目标用户：** 所有人——视觉是最通用的AI能力扩展

**与主机交互：**
- 插入时屏幕显示📷取景框预览，AI说："我能看见了！对准你想让我看的东西"
- 可以在屏幕上实时显示画面，或AI分析后显示标注结果

## 5. 🎛️ IR Blaster Module / 红外遥控模块

**一句话：** AI能用红外线控制你家所有传统家电——空调、电视、风扇。

**核心硬件：**
- 大功率红外发射LED × 3（940nm，多角度覆盖）
- IR接收头 VS1838B（用于学习遥控器信号）
- NPN三极管驱动电路

**AI怎么用它：**
- "太热了"→AI查当前温度（如有环境模块）→自动开空调调到26°C
- 学习模式：用户按遥控器，模块录制信号，AI记住"这是空调开机键"
- 离家时一句"我出门了"→AI关掉所有已学习的家电
- 定时场景："晚上10点把客厅灯调暗，电视关掉"
- 免去一桌子遥控器——对AI说话就行

**BOM成本：** ¥8-12

**3D打印难度：** 简单（小盒子，IR LED露出即可）

**目标用户：** 有传统家电的家庭（没有智能家居但想要语音控制）、老年人、租房族

**与主机交互：**
- 插入时屏幕显示遥控器图标，AI说："红外遥控就绪！把你的遥控器拿过来，我学一学"
- 学习完成后显示已学习设备列表（空调、电视、风扇……），一键发射

## 6. 🚨 Guardian Module / 安防守卫模块

**一句话：** AI成为你的家庭守卫——能感知人体活动并发出警报。

**核心硬件：**
- HC-SR501 PIR人体红外传感器（探测距离7m，120°角）
- 有源蜂鸣器（85dB）
- 可选：磁簧开关（门窗开闭检测）

**AI怎么用它：**
- 离家模式：检测到人体移动→AI先通过扬声器喊"请问是谁？"→无回应→蜂鸣器报警 + 推送通知到手机
- 老人独居看护：长时间无人体活动→AI主动关心："已经6小时没检测到活动，一切还好吗？"
- 结合视觉模块：有人进门→拍照识别→认识的人不报警，陌生人才警告
- 夜间模式：检测到起夜活动→不报警，但柔声提醒"小心走路"

**BOM成本：** ¥10-15

**3D打印难度：** 简单（PIR传感器需菲涅尔透镜开窗）

**目标用户：** 独居人士、有老人的家庭、小型办公室/工作室

**与主机交互：**
- 插入时屏幕显示🛡️图标，AI说："安防模块已就位。要我进入守卫模式吗？"
- 守卫中显示"Armed"状态 + 最后一次检测到活动的时间

## 7. 🌈 MoodLight Module / 氛围灯模块

**一句话：** AI掌握了光的艺术——用色彩表达信息、营造氛围。

**核心硬件：**
- WS2812B可编程RGB LED灯珠 × 16-30颗（灯带或灯环）
- 5V电源走RWC Bus（大电流需注意，可能需外接USB供电口）

**AI怎么用它：**
- 番茄钟：25分钟工作渐变绿→黄→红，时间到灯闪烁
- 天气预报可视化：蓝色=下雨，橙色=晴天，白色闪烁=下雪
- 通知指示：收到重要邮件闪一下蓝光，日历提醒闪黄光
- 音乐律动模式：配合主机麦克风拾音，灯光跟着音乐跳动
- 环境补光："现在光线太暗了，我帮你开个暖白光"

**BOM成本：** ¥15-25

**3D打印难度：** 中等（灯带导光条/扩散罩设计影响效果）

**目标用户：** 桌面美化爱好者、专注力训练、程序员/创意工作者

**与主机交互：**
- 插入时灯带彩虹流水一圈，AI说："氛围灯上线！你想要什么颜色？"
- 屏幕显示当前灯光模式 + 颜色选择器

## 8. 🧲 Finger Module / 机械手指模块

**一句话：** AI长出了一根"手指"——能按下物理按钮。

**核心硬件：**
- SG90微型舵机 × 1（扭矩1.8kg·cm）
- 3D打印按压臂/手指
- 可选：微动开关反馈（确认是否按到位）

**AI怎么用它：**
- 按咖啡机按钮："早上好！我帮你按了咖啡机，3分钟后好"
- 按台灯开关（对准物理按钮固定好）
- 定时按任何物理按钮——将"哑"设备接入AI控制
- 配合主机对话："帮我开咖啡机" → 舵机按下 → "已经按了，听到咖啡机启动的声音了"
- 猫咪逗猫棒模式（来回摆动）

**BOM成本：** ¥8-12

**3D打印难度：** 中等（需要可调节固定夹具，适配不同按钮位置）

**目标用户：** 极客/Maker、想把非智能设备变智能的人、懒人自动化

**与主机交互：**
- 插入时屏幕显示🤖手指动画，AI说："机械手指就位！告诉我要按什么？"
- 屏幕显示手指状态（收回/伸出）+ 操作记录

## 9. 📡 LoRa Beacon Module / 远程信标模块

**一句话：** AI获得了公里级无线通信能力——在没有WiFi的地方也能联系你。

**核心硬件：**
- Ra-01SC LoRa模块（SX1268，433MHz，通信距离3-5km空旷）
- SMA天线接口 + 弹簧天线
- 可选：GPS模块 ATGM336H（定位+授时）

**AI怎么用它：**
- 户外场景：主机带着LoRa模块放在营地，另一个LoRa节点带在身上→AI可以在3km内发短消息给你
- 宠物追踪：LoRa信标挂在狗项圈上，AI报告大致方向和距离
- 农场/大面积场地：在远处放传感器节点，通过LoRa回传数据给主机
- 应急通信：停电断网时仍可通过LoRa网络发信号

**BOM成本：** ¥20-30（不含GPS），¥35-50（含GPS）

**3D打印难度：** 简单（小盒子+天线孔）

**目标用户：** 户外爱好者、农场/庄园主、Maker/IoT玩家

**与主机交互：**
- 插入时屏幕显示📡信号图标，AI说："远程通信模块已连接，信号强度良好，可通信范围约Xkm"
- 显示已发现的LoRa节点列表 + 信号强度 + 最后通信时间

## 10. 🔌 PowerBank Module / 电源扩展模块

**一句话：** AI摆脱电线束缚——随身携带，户外也能用。

**核心硬件：**
- 18650锂电池 × 2（总容量约5000-7000mAh）
- IP5306 充放电管理芯片（集成升压+充电+电量指示）
- USB-C充电口
- 可选：5V/6V太阳能板接口（通过CN3791 MPPT充电）

**AI怎么用它：**
- 让主机变成便携设备，带去露营/野餐/户外活动
- AI管理电量："还剩40%电量，按当前使用强度还能撑3小时"
- 低电量时AI自动降低屏幕亮度、减少检测频率以延长续航
- 太阳能模式："阳光不错，正在太阳能充电，预计2小时充满"
- 搭配其他模块使用：户外=电源+环境感知+GPS，完整野外AI站

**BOM成本：** ¥30-50（不含电池），电池另计约¥20-30

**3D打印难度：** 中等（电池仓需结构强度，散热设计，USB-C开口精度）

**目标用户：** 户外用户、经常移动使用主机的人、阳台/花园场景

**与主机交互：**
- 插入时屏幕显示🔋电量百分比，AI说："电源模块已接入，当前电量XX%，我们可以出门了！"
- 常驻显示电池图标 + 剩余时间估算

## RWC Bus 引脚分配建议（8pin）

| Pin | 功能 | 说明 |
|-----|------|------|
| 1 | VCC 5V | 主电源 |
| 2 | GND | 地 |
| 3 | SDA (I2C) | 数据线 |
| 4 | SCL (I2C) | 时钟线 |
| 5 | UART TX | 串口发送（主机→模块） |
| 6 | UART RX | 串口接收（模块→主机） |
| 7 | GPIO / PWM | 通用IO/舵机控制 |
| 8 | ID / 1-Wire | 模块识别（DS2401或电阻分压识别） |

> Pin 8用于即插即用识别：每个模块内置唯一ID，主机插入后自动识别模块类型并加载对应驱动/UI。
