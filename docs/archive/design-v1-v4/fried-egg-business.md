## 1. 目标用户画像

### 画像A：科技潮玩青年（25-35岁）
- **谁**：一线城市互联网/设计从业者，桌面上已经有机械键盘、氛围灯、手办
- **在哪看到**：小红书种草帖、即刻社区、Twitter/X上的indie hacker圈
- **购买决策**：「造型太可爱了」+「这东西居然能跑AI」→ 冲动下单
- **放在哪**：显示器旁边，和手办、绿植并排，是桌面的「第四件摆件」
- **价格敏感度**：中低。¥299-499无脑买，超过¥599要犹豫

### 画像B：AI/开发者极客（20-40岁）
- **谁**：玩过树莓派、Home Assistant、喜欢折腾的技术人
- **在哪看到**：GitHub trending、Hacker News、YouTube拆解视频
- **购买决策**：「开源+模块化」→ 这是个能一直玩下去的平台
- **放在哪**：工作台上，接着各种线和传感器，旁边是焊台和示波器
- **价格敏感度**：看性价比。关注芯片规格、扩展能力，愿意为可玩性付溢价

### 画像C：礼物买家 / 非技术用户（22-45岁）
- **谁**：想送男/女朋友、同事一个有趣又实用的桌面小物
- **在哪看到**：抖音/TikTok短视频「我桌上的AI小蛋」、朋友圈晒图
- **购买决策**：「好可爱！送人绝对有面子」+「不贵」
- **放在哪**：床头柜、书桌、公司工位——哪里显眼放哪里
- **价格敏感度**：高。¥299是心理上限，最好¥199-249

## 2. 开箱体验设计

### 包装
- **外盒**：正方形，紧凑。白色卡纸+烫银logo，侧面一句话：`Your AI, sunny side up.`
- **尺寸**：约 12×12×6cm，比AirPods盒大两圈，拿在手里有份量但不笨重
- **风格**：Apple级克制。没有花哨的参数罗列，只有产品剪影和一句slogan
- **环保**：全纸质内衬（非塑料），模块化产品从包装就传达「可持续」

### 开箱序列
1. **掀开盖子**：煎蛋正面朝上，嵌在纸托里，屏幕贴着一张半透明保护纸，上面印着一个 🍳 emoji
2. **拿起煎蛋**：手感出乎意料地好（重量约80-120g，有质感不廉价）
3. **底下一层**：Type-C线 + 快速开始卡片（一张，正反面，不是说明书）
4. **插线，3秒亮屏**：圆形屏幕播放一个煎蛋「滋滋」煎熟的开机动画（2-3秒）
5. **扫码配对**：屏幕显示二维码，手机扫一下就连上

### 时间线
| 步骤 | 时间 |
|------|------|
| 开箱到拿出设备 | 10秒 |
| 插线到亮屏 | 3秒 |
| 扫码到配对完成 | 30秒 |
| **总计：从拆封到可用** | **< 1分钟** |

### "值了"时刻
- **第一次**：开机动画——蛋黄屏幕上煎蛋滋滋作响，用户会笑出来
- **第二次**：对它说第一句话，它回应了，圆形屏幕上表情变化——「这东西活的！」
- **第三次**：拍照发朋友圈/小红书，收到第一条评论「这是什么！在哪买！」

## 3. 桌面场景

### 感觉
放在桌上像一个**安静的小伙伴**。不是冰冷的电子设备，是一个有温度的「桌面宠物」。蛋白的有机曲线在直线条的办公设备中非常跳脱，自然成为视觉焦点。

### 典型桌面构图
```
[显示器]
[键盘]  🍳  [咖啡杯]
        [鼠标垫]
```
煎蛋在键盘右侧或显示器底座旁，占地面积约一个马克杯底部大小。

### 尺寸建议
- **整体（含蛋白）**：约 **90×80mm**（长轴×短轴），不规则椭圆
- **圆形屏幕（蛋黄）**：直径约 **40-45mm**（1.7-1.8寸），偏心放置
- **厚度**：约 **22-25mm**
- **参考**：比一个鸡蛋大一圈，比Echo Dot小一半，和AirPods Max的耳罩差不多大

### 角度
- **默认微倾15°**：底部设计成自然后仰，屏幕面向用户，不需要支架
- **可选磁吸底座**（另售）：提供30°/60°两档，适配显示器侧面吸附
- **平放也OK**：当桌面装饰/氛围灯，屏幕朝天花板也有趣（显示天气、动态壁纸）

## 4. 社交传播

### 最上镜的造型
- 45°俯拍，煎蛋旁边放一杯真咖啡 → 「我的早餐搭档」
- 屏幕显示可爱表情 + 窗边自然光 → 治愈系
- 多个煎蛋排列（不同表情）→ 产品全家福

### 必须分享的Feature
1. **表情系统**：蛋黄屏幕上的动态表情（眨眼、打哈欠、被戳会生气）→ 用户会录屏
2. **煎蛋煎熟开机动画**：每次开机都想录一次
3. **模块进化**：第一次装上新模块，蛋蛋「长出手/脚」的动画 → 里程碑时刻必发
4. **自定义蛋黄脸**：用户画的表情 → UGC内容爆发

### 竞品对标分析

| 竞品 | 价格 | 我们的优势 |
|------|------|-----------|
| Divoom Ditoo ¥399 | 像素屏蓝牙音箱 | 我们有AI+模块化，不只是音箱 |
| 天猫精灵 ¥89 | 纯语音助手 | 我们有屏幕+造型+开源生态 |
| Echo Dot $50 | 语音助手 | 我们有个性、有进化路径、有社区 |
| Tamagotchi Uni ¥399 | 电子宠物 | 我们是真AI，不是脚本 |

### 甜蜜点：¥249 基础版

**理由：**
- **< ¥300**：冲动消费阈值。不需要「跟老婆商量」
- **> ¥199**：不会被当成廉价玩具，有品质感
- **¥249**：在小红书种草帖里，「不到250」是一个非常好的话术锚点
- **模块升级路径**：基础版低价引流，后续模块（Audio ¥99、Motor ¥149、Camera ¥129）持续变现
- **毛利**：目标BOM ¥80-100，毛利率60%+（硬件引流，生态赚钱）

### 首发策略
- 众筹早鸟价 **¥199**（限1000台）→ 制造稀缺+口碑种子
- 标准零售 **¥249**
- 礼盒版（Starter Kit + 定制贴纸）**¥399** → 送礼场景

## 6. 造型建议（商业/用户视角）

### 尺寸
- **蛋白外轮廓**：90×80mm，有机不规则形（不要太规则，要像真煎蛋那样随意）
- **蛋黄屏幕**：∅40-45mm，偏心位置（不要正中央，偏左上更自然）
- **厚度**：22-25mm（比手机厚一倍，拿在手里有「鹅卵石」感）
- **重量**：80-120g（要有「密度感」，太轻会廉价）

### 颜色策略
| 版本 | 蛋白 | 蛋黄（屏幕关闭时） | 定位 |
|------|------|-------------------|------|
| **经典白** | 哑光暖白 | 屏幕自发光 | 首发主力 |
| **太阳蛋** | 亮面纯白 | 橙黄边框 | 限定色 |
| **夜煎蛋** | 哑光深灰 | 屏幕自发光 | 科技感 |
| **抹茶蛋** | 哑光薄荷绿 | 屏幕自发光 | 联名/季节限定 |

**首发只出经典白**。一个SKU降低库存风险，白色最上镜，后续颜色制造持续话题。

### 质感
- **蛋白**：哑光磨砂，类baby skin触感（类似Google Pixel手机后盖）→ 拿起来就不想放下
- **边缘**：蛋白边缘微微翘起，模拟真煎蛋的焦边 → 细节控会爱死
- **底部**：硅胶防滑垫，微微内凹 → 放桌上纹丝不动

### 加分细节（按优先级排序）

1. **🥇 蛋白边缘呼吸灯**（必须有）
   - 通知来了柔和亮起，像蛋白在发光
   - 充电时缓慢呼吸，有生命感
   - 可自定义颜色 → 氛围灯功能免费送

2. **🥈 磁吸模块接口**（核心卖点）
   - 底部/侧面磁吸触点，吸上去「咔哒」一声
   - 这个声音要设计好 → ASMR级满足感
   - 每装一个模块，屏幕播放进化动画

3. **🥉 可挂可立可吸**
   - 底部磁铁 → 吸冰箱、显示器边框、白板
   - 背面挂钩槽 → 挂墙上当智能相框
   - 默认就能立 → 不需要额外配件

4. **贴纸系统**
   - 附赠一张煎蛋主题贴纸（蛋壳碎片、小鸡、平底锅）
   - 蛋白表面留有「贴纸友好区」→ 鼓励个性化
   - 用户贴完拍照分享 → 免费UGC

5. **触感反馈**
   - 戳蛋黄屏幕，轻微震动 → 像戳真蛋黄
   - 这个细节会被写进每一篇测评

## 7. 商业总结

### 一句话定位
> **煎蛋是你桌上第一个有表情的AI伙伴，也是一个会长大的蛋蛋机器人的起点。**

### 核心商业逻辑
```
低价可爱硬件引流（¥249）
    → 用户晒图自传播（0成本获客）
        → 模块持续购买（LTV ¥500-800）
            → 开发者生态（第三方模块/应用）
                → 平台效应
```

### 第一年目标（建议）
| 指标 | 目标 |
|------|------|
| 众筹首月销量 | 3,000-5,000台 |
| 第一年总销量 | 20,000-30,000台 |
| 小红书/社交曝光 | 500+篇UGC |
| 模块attach rate | 40%用户买≥1个模块 |

### 风险与对策
| 风险 | 对策 |
|------|------|
| 「可爱但没用」 | 确保Day 1有3个killer use case（时钟/天气、AI对话、通知中心） |
| 屏幕太小看不清 | 不靠信息密度，靠表情和动画取胜 |
| 模块生态起不来 | 首发自己出3个核心模块，开源SDK让社区跟进 |
| 价格战 | 不打价格战，打情感战——没人会山寨「一个会长大的蛋」 |
