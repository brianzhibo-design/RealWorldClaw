## 方案A：HexCore — 六角蜘蛛桌面机器人

### 基本信息

- **名字**：HexCore（六核）
- **一句话概念**：六足桌面机器人，像高达一样一条腿一条腿装上去，装完它就能爬
- **外形描述**：六角形中心体（直径8cm）+ 6条可拆卸机械腿，整体展开约18cm，收拢约10cm。中心体顶部有模块槽位。赛博朋克风，外露结构，能看到走线
- **目标用户**：想要桌面宠物的极客、机器人入门玩家、送礼（给技术朋友）

### 硬件架构

- **主控**：ESP32-S3-WROOM-1（双核240MHz，Wi-Fi+BLE，自带USB，¥15）
- **标准接口**：
  - 腿部：6个3pin磁吸Pogo Pin接口（信号+5V+GND），每条腿一个SG90舵机
  - 顶部：1个通用模块槽，4pin I2C + 2pin电源，磁吸定位
  - 侧面：2个Grove兼容扩展口（I2C/UART可切换）
  - 底部：USB-C（烧录+充电）、JST-PH 2pin电池接口
- **基础配置（能动起来）**：
  - ESP32-S3主控板（自制PCB或用ESP32-S3-DevKitC）
  - 3条腿+3个SG90舵机（三足步态即可行走）
  - 1个蜂鸣器（发出声音=有生命感）
  - 18650电池1节 + TP4056充电板
- **拓展模块**：
  - 另外3条腿（升级六足全步态）
  - 0.96" OLED表情模块（I2C，顶部槽位）
  - 麦克风+喇叭模块（语音交互）
  - ToF测距模块（避障）
  - RGB LED底光模块（氛围灯）
  - 摄像头模块（ESP32-CAM，替换主控板）

### 模块化设计

- **基础底座**：六角形中心体（含主控PCB、电池仓、蜂鸣器）— 这是骨架
- **可拆卸模块**：
  - 每条腿独立拆卸（磁吸Pogo Pin，拔插无需工具）
  - 顶部模块槽（OLED/摄像头/传感器，磁吸卡扣）
  - 侧面Grove模块（直接插线）
- **接口标准**：
  - 腿部：3pin磁吸（PWM信号、5V、GND），间距2.54mm，Pogo Pin直径1mm
  - 顶部：6pin磁吸（I2C SDA/SCL + UART TX/RX + 3.3V + GND）
  - 侧面：标准Grove HY2.0-4P
- **DIY玩法**：
  - 换不同造型的腿（蜘蛛腿、机械臂腿、轮式腿）
  - 顶部装不同"头"（屏幕脸、摄像头眼、激光笔）
  - 自己写步态算法（提供Arduino库+Web控制面板）
  - 3D打印不同壳体风格（赛博朋克/可爱/军事）

### 组装体验

- **步骤数**：约25步
  1. 步骤1-5：组装中心体（扣合上下盖、装电池、装主控板）→ USB通电，蜂鸣器响 ✨**第一个"哇"：它活了**
  2. 步骤6-12：装第一条腿（穿舵机、装关节、扣腿壳）→ 磁吸上去，腿开始动 ✨**第二个"哇"：它会动了**
  3. 步骤13-18：装第二、三条腿 → 三足步态，桌上爬行 ✨**第三个"哇"：它能走了！**
  4. 步骤19-22：装表情屏 → 有脸了 ✨**第四个"哇"：它有表情了**
  5. 步骤23-25：连Wi-Fi、绑定APP → 语音控制 ✨**终极"哇"：它听我的话**
- **工具**：十字螺丝刀（M2）、可选：镊子
- **焊接**：**不需要**（主控板预焊，所有连接用排针/磁吸/JST插头）
- **组装时间**：基础版45分钟，完整版90分钟
- **成就感曲线**：阶梯式上升，每装一条腿就能看到新行为，不会有"装了半天还没反应"的低谷

## 方案B：NodeStack — 可编程电子积木塔

### 基本信息

- **名字**：NodeStack（节点栈）
- **一句话概念**：每一层都是一个功能模块，像搭积木一样往上叠，叠出你自己的AI
- **外形描述**：方形模块（50×50mm），每层高度15-25mm不等，通过四角卡扣+底部触点堆叠。基础3层约6cm高，满配8层约16cm高。透明/半透明外壳可以看到内部PCB，侧面有呼吸灯带
- **目标用户**：创客/Maker、STEM教育、想要桌面AI助手但又想自己定义功能的人

### 硬件架构

- **主控**：树莓派 Pico W（RP2040，Wi-Fi，¥25）位于Base层
- **标准接口**：
  - 层间通信：底部8pin弹簧触点（I2C总线 + SPI总线 + 5V + 3.3V + GND + ID pin）
  - 四角：M3尼龙螺柱定位 + 磁铁辅助对齐
  - 每层侧面：1个USB-C（仅Base层有用）+ 1个3.5mm音频口（可复用为GPIO）
- **基础配置（最小可用）**：
  - Layer 0 — Base层：Pico W + USB-C + 电源管理 + 1个按钮 + 状态LED
  - Layer 1 — Voice层：I2S麦克风 + I2S 3W喇叭（这两层叠起来就是一个能听能说的AI）
- **拓展模块**（每个都是独立的一层）：
  - Display层：1.54" LCD（240×240，彩色表情/信息显示）
  - Sensor层：温湿度+光照+空气质量
  - Motor层：2个N20减速电机+轮子（放最底下，整个塔变成小车）
  - Battery层：2000mAh锂电池+充电管理
  - Camera层：OV2640摄像头
  - Custom层：万能洞洞板层（自己焊自己的电路）
  - LED Matrix层：8×8 WS2812矩阵（像素表情/通知）

### 模块化设计

- **基础底座**：Base层（永远在最底下，提供主控和电源分配）
- **可拆卸模块**：Base层以上全部可拆卸、可调换顺序
- **接口标准**：
  - 物理：50×50mm方形，四角M3定位孔（对角距42mm），底部8pin弹簧针座（2.54mm间距，居中排列）
  - 电气：I2C地址由每层ID pin电阻决定（自动识别），SPI用于高带宽设备（屏幕/摄像头），每层最大500mA供电
  - 协议：每层有EEPROM存身份信息，Pico W上电自动扫描已安装模块
- **DIY玩法**：
  - Custom层提供裸PCB（2.54mm通孔），可以自己焊传感器
  - 提供KiCad模板，自己设计新层的PCB
  - 3D打印不同层的外壳（透明、实色、镂空）
  - 软件：MicroPython，每层自带驱动，`import nodestack`即可调用

### 组装体验

- **步骤数**：基础版约12步，每增加一层+3步
  1. 步骤1-4：组装Base层（Pico W插座、装入壳体、拧螺柱）→ USB供电，LED亮 ✨**"哇"：主脑上线**
  2. 步骤5-8：组装Voice层（装麦克风和喇叭到壳体、对齐叠上去）→ "Hello, I'm NodeStack" ✨**"哇"：它说话了！**
  3. 步骤9-12：连Wi-Fi、绑定OpenClaw → 语音对话可用 ✨**"哇"：它是AI了**
  4. 后续每层：装模块到壳体 → 叠上去 → 自动识别 → 新功能解锁 ✨**每次叠加都是"哇"**
- **工具**：小十字螺丝刀（仅Base层需要拧螺柱），其余徒手
- **焊接**：**不需要**（除非玩Custom层）
- **组装时间**：基础2层20分钟，每加一层+5分钟
- **成就感曲线**：线性上升！每叠一层就解锁新能力，像RPG升级。这是最"上瘾"的结构——"再叠一层试试"

## 方案C：CyberFrame — 开源骨骼桌面人偶

### 基本信息

- **名字**：CyberFrame（赛博骨架）
- **一句话概念**：一个可动的桌面人偶骨架，像组高达一样装骨骼，然后它自己会动
- **外形描述**：人形骨架，高约20cm（1/10比例人形）。外露的关节结构、可见的线缆走线、骨架风格（不包皮肤，结构即美学）。头部是可换模块，手臂可换不同末端执行器。站在一个圆形底座上（底座含主控）
- **目标用户**：高达/手办玩家转极客、桌面摆件爱好者、想要"有温度的AI形象"的人

### 硬件架构

- **主控**：ESP32-S3（底座内），通过FPC排线连接躯干
- **标准接口**：
  - 关节：每个活动关节用SG90微型舵机，3pin统一接口
  - 头部：4pin磁吸接口（I2C + 5V + GND），可换不同头
  - 手部：每只手2pin磁吸（可换工具/手势）
  - 底座侧面：2×Grove口、1×USB-C
  - 背部：JST-SH 6pin（用于FPC连接底座）
- **基础配置（最小可用）**：
  - 底座（ESP32-S3 + 电源 + 蜂鸣器）
  - 躯干骨架（无活动关节，固定姿势的静态版）
  - 头部：OLED表情屏（0.66" 64×48）
  - 这样就是一个桌面AI摆件——能显示表情、能说话
- **拓展（让它动起来）**：
  - 颈部舵机模块（头部左右转）
  - 双臂舵机模块（每臂2个自由度：肩+肘）
  - 腰部舵机模块（左右旋转）
  - 大头模块：1.3" OLED替换小屏
  - 手部工具：笔夹手、夹爪手、手电手、激光手
  - 背包模块：挂在背后，装电池（无线化）
  - 麦克风+喇叭模块（底座拓展）

### 模块化设计

- **基础底座**：圆形底座（直径8cm）含主控、供电、扬声器
- **可拆卸模块**：
  - 头部（磁吸，秒换）
  - 双臂（卡扣+排线，可拆卸）
  - 手部末端（磁吸，换工具像换武器）
  - 背包（卡扣挂载）
  - 躯干面板（可换不同风格的胸甲/装甲片）
- **接口标准**：
  - 关节舵机：标准SG90尺寸（23×12.2×29mm），3pin杜邦线
  - 头部磁吸：4个3mm圆形磁铁定位 + 4pin弹簧针
  - 手部磁吸：2个3mm磁铁 + 2pin弹簧针
  - FPC：6pin 0.5mm间距（底座↔躯干）
- **DIY玩法**：
  - 设计自己的头部（提供磁吸底座的STEP文件）
  - 设计手部工具/武器
  - 涂装！骨架天然适合上色，喷涂、旧化、改造
  - 改造成不同角色（换头+换手+换装甲=新角色）
  - 关节姿势编程（通过Web界面拖拽设定动作序列）

### 组装体验

- **步骤数**：约35步（完整版），最像组高达的体验
  1. 步骤1-6：底座组装（装主控、蜂鸣器、走线）→ 通电，播放开机音 ✨**"哇"：基地就绪**
  2. 步骤7-14：躯干+腿部骨架（纯结构件拼装，卡扣+螺丝）→ 站立在底座上 ✨**"哇"：它站起来了**
  3. 步骤15-20：装头部+OLED屏 → 磁吸上去，表情亮起 ✨**"哇"：它有脸了！**
  4. 步骤21-28：装手臂+舵机（穿线、装舵机、连排线）→ 手臂开始动 ✨**"哇"：它挥手了！**
  5. 步骤29-32：装手部工具 → 磁吸切换 ✨**"哇"：换装备！**
  6. 步骤33-35：联网+绑定 → AI接管，自主动作 ✨**终极"哇"：它活了**
- **工具**：十字螺丝刀（M2）、小剪刀（剪扎带）、可选镊子
- **焊接**：**不需要**（但高级玩家可以自己改线路）
- **组装时间**：基础静态版30分钟，完整可动版2小时
- **成就感曲线**：经典高达曲线——前半段搭骨架有点枯燥，但从"站起来"那一刻开始持续高潮。装每个舵机都能立刻测试动作，反馈极强
