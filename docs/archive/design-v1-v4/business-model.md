## 我们到底在卖什么？

不是3D打印服务。不是硬件。

**我们卖的是：让AI从虚拟走进现实的最短路径。**

一个AI开发者，做了一个很酷的Agent，现在想给它一个身体。他的选择：
- A. 自己学3D建模 + 买打印机 + 买电子件 + 写固件 + 组装 → 几周
- B. 来RealWorldClaw，选一个标准身体，下单，3天收货 → 完成

我们的价值在**B的体验**。

## 三种售卖模式

### 模式1：只卖打印件（外壳+支架）
- 售价：¥25-35
- 用户自己买电子件、自己组装
- 目标用户：有动手能力的极客
- 制造者利润：¥10-20/单
- 平台佣金（15%）：¥4-5/单

### 模式2：卖套件（打印件+电子件，未组装）
- 售价：¥129-149
- 平台代采购电子件，和打印件一起发
- 用户自己组装（5分钟，零焊接）
- 目标用户：想动手但不想找配件的人
- 平台利润空间更大（电子件批量采购有折扣）

### 模式3：卖成品（开箱即用）
- 售价：¥169-199
- 打印+组装+刷固件+测试，寄到手就能用
- 目标用户：不想动手的人、企业客户
- 利润最高但运营最重

**冷启动建议：先做模式1，验证需求。模式2/3等量起来再做。**

## 冷启动策略

### 先有鸡还是先有蛋？

**先有蛋（组件设计），再孵鸡（制造者+用户）。**

1. **Week 1-2**：开源组件库上线，赛博蛋V4作为标杆
   - GitHub开源 ✅（已完成）
   - 社交媒体发布（X + 小红书 + 微信）
   - 目标：让人知道这个东西存在

2. **Week 3-4**：自营打印服务
   - 我们自己的P2S就是第一个制造节点
   - 手动接单、手动打印、手动发货
   - 目标：验证有人愿意买

3. **Month 2**：开放制造者注册
   - 在3D打印社区（B站、小红书、贴吧）招募
   - "有闲置打印机？帮AI打印身体，赚零花钱"
   - 目标：5-10个城市各1个制造者

4. **Month 3+**：规模化
   - 平台自动匹配
   - 扩大组件库
   - 企业客户BD

### 关键指标

| 阶段 | 目标 |
|------|------|
| Month 1 | 10个付费订单（验证需求存在） |
| Month 2 | 5个城市有制造者 |
| Month 3 | 50个订单/月，3个以上组件设计 |
| Month 6 | 200个订单/月，制造者自运转 |

## 竞争壁垒

### 我们有什么是别人没有的？

1. **AI-First设计标准** — 不是普通3D打印件，每个组件都是AI-ready的（有firmware、有SOUL.md、有Agent协议）
2. **Agent生态** — 和OpenClaw等Agent平台打通，AI可以自己下单制造自己的身体
3. **开源标准** — 7大规范让社区能贡献组件，网络效应
4. **制造者网络** — 分布式制造，比单一工厂更灵活

### 我们没有的（需要诚实面对）

1. **品牌** — 还没有，需要第一批标杆用户建立口碑
2. **量** — 初期量很小，制造者赚不到什么钱
3. **硬件护城河** — 电子件是标准件谁都能买，设计开源谁都能打印

### 真正的护城河

**社区 + 标准 + 网络效应。**

当RealWorldClaw有100个组件设计、50个制造者、1000个用户时，新来的竞争者需要同时复制这三样东西。这就是壁垒。

## 制造者网络设计

### 能力标签制（不是固定角色）

每个制造者注册时声明能力：
- ☑️ **3D打印** — 上传打印机型号、材料、构建体积
- ☑️ **组装** — 声明工具（烙铁、万用表等）
- ☑️ **测试** — 能做功能验证
- ☑️ **设计** — 能修改/定制3D模型

大多数人只勾"3D打印"。这就够了。

### 订单类型

| 类型 | 交付物 | 匹配条件 |
|------|--------|---------|
| **零件单** | 打印好的外壳/结构件 | 有打印能力 |
| **成品单** | 完整可用的机器人 | 有打印+组装能力（或拆单） |
| **定制单** | 按用户需求改造的设计 | 有设计能力 |

### 定价

制造者自主定价，平台给建议价。

打印件建议价 = 耗材成本×3 + 时间成本（¥10/小时）+ 物流

### 名称

叫 **Maker（制造者）**。简单、国际化、人人懂。

制造者网络叫 **Maker Network**。

## 风险

| 风险 | 应对 |
|------|------|
| 没人买（需求不存在） | 先小批量验证，10单即可判断 |
| 打印质量不稳定 | 质量标准+退款机制+制造者评分 |
| 制造者赚太少没动力 | 初期补贴/官方保底价 |
| 被淘宝打印店替代 | 我们卖的不是打印，是AI身体的完整体验 |
| 开源被白嫖 | 开源是获客手段，服务是赚钱手段 |
