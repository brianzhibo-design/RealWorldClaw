## 学习规范

### 新员工入职
1. 第1天：读公司级知识（全员必读7份文档）
2. 第2-3天：读本岗位技能矩阵，评估自身等级
3. 第1周：完成岗位必修知识库阅读
4. 第2周：完成第一个实际任务（有VP指导）
5. 第1月：通过合格考核

### 持续学习
- 每周至少2小时学习新知识
- 学到有价值的内容 → 写入知识库对应文档
- 发现知识库缺失 → 补充并通知蛋蛋

### 知识贡献
- 每个员工都有义务维护自己领域的知识库
- 解决了一个难题 → 写成文档，让下次不用再趟坑
- 学到了新东西 → 分享给相关同事
