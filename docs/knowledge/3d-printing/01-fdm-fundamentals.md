## 1. FDM工作原理（熔融沉积）

### 基本原理
FDM（Fused Deposition Modeling）即熔融沉积建模，通过加热塑料材料至熔融状态，逐层挤出构建3D物体。

**核心组件：**
- 热端（Hot End）：加热并熔化塑料丝
- 挤出机（Extruder）：推送材料进入热端
- 热床（Heat Bed）：提供第一层附着力
- 移动平台：实现X、Y、Z轴精确定位

**工作流程：**
1. 塑料丝被加热到熔融状态（通常190-330°C）
2. 熔融材料通过喷嘴挤出，形成细线
3. 每层打印完成后，平台下移一个层高
4. 重复此过程直到模型完成

### 何时选择FDM？
✅ **适合场景：**
- 功能性原型制作
- 大尺寸物体打印
- 材料选择多样性需求
- 成本敏感项目
- 机械强度要求较高

❌ **不适合场景：**
- 需要极高表面精度（如珠宝制作）
- 超精细细节（小于0.1mm）
- 透明或光学质量要求
- 大批量生产

## 2. 常见材料对比

### 2.1 入门级材料

#### PLA（聚乳酸）
**温度设置：** 喷嘴190-240°C，热床35-65°C

**特性：**
- ⭐⭐ 耐热性（50-60°C软化）
- ⭐⭐ 冲击强度
- ⭐⭐⭐ 拉伸强度
- 打印难度：极易

**何时使用：**
- 初学者练习
- 装饰性物品
- 原型验证
- 室内使用物品

**何时不使用：**
- 高温环境（>50°C）
- 需要高冲击强度
- 户外使用

#### PETG（改性聚对苯二甲酸乙二醇酯）
**温度设置：** 喷嘴240-270°C，热床60-80°C

**特性：**
- ⭐⭐⭐ 耐热性（80-85°C）
- ⭐⭐⭐ 冲击强度
- ⭐⭐ 拉伸强度
- 打印难度：容易

**何时使用：**
- 需要化学耐性
- 食品安全应用
- 透明或半透明效果
- 机械性能平衡需求

**何时不使用：**
- 极高温度环境
- 需要支撑的复杂结构（粘性强）

### 2.2 工程级材料

#### ABS（丙烯腈-丁二烯-苯乙烯共聚物）
**温度设置：** 喷嘴240-280°C，热床90-100°C

**特性：**
- ⭐⭐⭐ 耐热性（95-105°C）
- ⭐⭐⭐ 冲击强度
- ⭐⭐⭐ 拉伸强度
- 打印难度：中等（需要封闭仓体）

**何时使用：**
- 汽车零件
- 电子产品外壳
- 需要后处理（丙酮平滑）
- 机械应力环境

**何时不使用：**
- 开放式打印机
- 办公环境（有异味）
- 初学者

#### ASA（丙烯酸-苯乙烯-丙烯腈）
**温度设置：** 喷嘴240-280°C，热床90-100°C

**特性：**
- ⭐⭐⭐ 耐热性
- ⭐⭐⭐ 冲击强度
- ⭐⭐⭐ 拉伸强度
- 优秀的UV耐性

**何时使用：**
- 户外使用部件
- 汽车外饰件
- 需要UV稳定性

**何时不使用：**
- 室内装饰品
- 透明度要求高的场景

### 2.3 特殊功能材料

#### TPU（热塑性聚氨酯）
**温度设置：** 喷嘴200-250°C，热床30-45°C

**特性：**
- ⭐ 耐热性
- ⭐⭐⭐⭐⭐ 柔韧性
- ⭐ 拉伸强度
- 打印难度：困难（柔软材料）

**何时使用：**
- 手机壳、保护套
- 减震垫
- 密封件
- 柔性铰链

**何时不使用：**
- 结构强度要求高
- AMS自动换料系统
- 高精度细节

#### PA/Nylon（尼龙）
**温度设置：** 喷嘴260-290°C，热床100-120°C

**特性：**
- ⭐⭐⭐⭐ 耐热性
- ⭐⭐⭐ 冲击强度
- ⭐⭐⭐⭐ 拉伸强度
- 打印难度：困难（需要干燥）

**何时使用：**
- 齿轮、轴承
- 工业功能件
- 高强度应用

**何时不使用：**
- 湿度控制困难环境
- 快速原型需求

#### PC（聚碳酸酯）
**温度设置：** 喷嘴260-290°C，热床100-120°C

**特性：**
- ⭐⭐⭐⭐ 耐热性（140°C）
- ⭐⭐⭐⭐ 冲击强度
- ⭐⭐⭐⭐ 拉伸强度
- 打印难度：困难

**何时使用：**
- 高温应用
- 防弹级强度需求
- 光学部件

**何时不使用：**
- 开放式打印机
- 成本敏感项目

### 2.4 复合材料

#### PLA-CF（碳纤维增强PLA）
**温度设置：** 喷嘴210-240°C，热床45-65°C

**特性：**
- ⭐⭐ 耐热性
- ⭐⭐ 冲击强度
- ⭐⭐⭐⭐ 拉伸强度
- 优秀的刚性

**何时使用：**
- 无人机部件
- 轻量化结构件
- 需要高刚性但重量轻

**何时不使用：**
- 0.2mm喷嘴
- 普通钢材喷嘴（需要硬化钢）

## 3. 层高选择策略

层高是影响打印质量和时间的关键参数。

### 层高范围与应用

#### 0.08-0.12mm（超精细）
**何时使用：**
- 微型模型
- 需要极细表面纹理
- 珠宝原型
- 艺术品复制

**何时不使用：**
- 大尺寸物体（时间成本过高）
- 功能性部件
- 快速原型

**影响：** 表面质量最佳，打印时间最长，层间结合稍弱

#### 0.15-0.2mm（精细）
**何时使用：**
- 精密零件
- 手办模型
- 需要平滑表面的功能件
- 小批量成品

**何时不使用：**
- 纯功能性部件
- 时间敏感项目

**影响：** 表面质量与时间的良好平衡

#### 0.25-0.3mm（标准）
**何时使用：**
- 功能性原型
- 日常使用物品
- 机械零件
- 大多数应用场景

**何时不使用：**
- 需要极高表面质量
- 微小细节重要

**影响：** 时间与质量的最佳平衡，机械强度良好

#### 0.32mm及以上（粗糙）
**何时使用：**
- 大尺寸打印
- 纯功能测试
- 快速概念验证
- 结构强度优先

**何时不使用：**
- 表面质量重要
- 精密配合
- 展示用途

**影响：** 最快打印速度，最强层间结合，表面纹理明显

### 层高与喷嘴关系
- **0.4mm喷嘴：** 建议0.1-0.3mm层高
- **0.6mm喷嘴：** 建议0.2-0.4mm层高
- **0.8mm喷嘴：** 建议0.3-0.6mm层高

**规则：** 层高不应超过喷嘴直径的75%

## 4. 填充率和填充图案选择

填充影响强度、重量、打印时间和材料消耗。

### 4.1 填充率选择

#### 0-15%（极低）
**何时使用：**
- 装饰品
- 灯罩、花瓶
- 快速原型
- 材料节约

**何时不使用：**
- 需要承受载荷
- 顶面质量要求高

#### 15-30%（低）
**何时使用：**
- 一般功能件
- 家居用品
- 展示模型

**何时不使用：**
- 高强度需求
- 冲击载荷

#### 40-60%（中等）
**何时使用：**
- 工具和夹具
- 机械零件
- 日常功能件

**何时不使用：**
- 极限强度需求
- 成本敏感

#### 70-100%（高）
**何时使用：**
- 承重结构
- 齿轮、轴承
- 高应力环境

**何时不使用：**
- 材料成本考量
- 快速原型

### 4.2 填充图案对比

#### Grid/Grid（网格）
**特点：**
- 简单快速
- 各向均匀强度
- 材料效率高

**何时使用：**
- 通用应用
- 快速原型
- 成本敏感

**何时不使用：**
- 需要最佳强重比
- 复杂应力分布

#### Gyroid（螺旋）
**特点：**
- 优秀的强度重量比
- 各向同性
- 良好的顶面支撑

**何时使用：**
- 航空航天应用
- 需要最佳机械性能
- 轻量化需求

**何时不使用：**
- 极简打印时间要求
- 老旧打印机（计算压力大）

#### Honeycomb/3D Honeycomb（蜂窝）
**特点：**
- 传统优秀强重比
- 类似自然蜂窝结构
- 材料效率高

**何时使用：**
- 结构工程
- 航空模型
- 承重平板

**何时不使用：**
- 各向同性强度需求
- 打印速度优先

#### Lightning（闪电）
**特点：**
- 极快打印速度
- 最少材料消耗
- 树状支撑结构

**何时使用：**
- 装饰品
- 快速原型
- 材料节约优先
- 顶面质量不重要

**何时不使用：**
- 任何强度需求
- 精密部件

#### Concentric（同心圆）
**特点：**
- 优美的截面
- 各层强度一致
- 适合圆形物体

**何时使用：**
- 圆形容器
- 装饰品
- 柔性材料

**何时不使用：**
- 矩形物体
- 高强度需求

### 填充图案选择决策树
```
装饰品/快速原型 → Lightning (0-15%)
通用功能件 → Grid/Cubic (20-40%)
高强度需求 → Gyroid/Honeycomb (40-80%)
航空/轻量化 → Gyroid (15-30%)
圆形物体 → Concentric (20-60%)
```

## 5. 壁厚与壁圈数选择

壁厚直接影响强度、表面质量和打印时间。

### 5.1 壁圈数基础

#### 1-2圈
**何时使用：**
- 装饰品
- 灯罩等薄壁结构
- 材料节约

**何时不使用：**
- 功能性部件
- 需要承载

**强度：** ⭐⭐
**适用场景：** 展示、装饰

#### 3-4圈（推荐）
**何时使用：**
- 大多数应用
- 功能性原型
- 日常用品

**何时不使用：**
- 极限强度需求
- 超薄壁要求

**强度：** ⭐⭐⭐⭐
**适用场景：** 通用最佳选择

#### 5圈以上
**何时使用：**
- 高强度要求
- 螺纹连接
- 重载应用

**何时不使用：**
- 成本敏感
- 快速打印
- 薄壁结构

**强度：** ⭐⭐⭐⭐⭐
**适用场景：** 工程应用

### 5.2 壁厚与喷嘴关系

**0.4mm喷嘴：**
- 1圈 = 0.4mm
- 2圈 = 0.8mm
- 3圈 = 1.2mm（推荐最小功能壁厚）
- 4圈 = 1.6mm（通用推荐）

**设计原则：**
- 壁厚应为挤出宽度的整数倍
- 最小功能壁厚：3个挤出宽度
- 螺纹孔推荐：至少5个挤出宽度

### 5.3 特殊考量

#### 薄壁打印（<0.8mm）
**技巧：**
- 使用花瓶模式（螺旋外轮廓）
- 降低打印速度
- 使用较低温度

**何时使用：**
- 艺术品
- 容器
- 灯罩

#### 超厚壁（>5mm）
**考虑：**
- 增加填充率而非壁厚
- 注意冷却收缩
- 可能产生内应力

## 6. 首层附着力优化

首层是成功打印的关键，关系到整个打印的成败。

### 6.1 热床温度设置

#### PLA系列
- **Cool Plate：** 35-45°C
- **Engineering Plate：** 45-65°C + 胶棒
- **PEI Plate：** 45-65°C
- **高于45°C时：** 建议开前门/移除顶盖

#### PETG系列
- **不推荐Cool Plate**
- **Engineering/PEI Plate：** 60-80°C + 胶棒
- **高于70°C时：** 建议开前门/移除顶盖

#### ABS/ASA
- **Engineering/PEI Plate：** 90-100°C + 胶棒
- **保持封闭仓体**

### 6.2 首层设置优化

#### 首层层高
- **标准：** 0.2-0.3mm（增加接触面积）
- **精细打印：** 可设置为正常层高的125-150%

#### 首层速度
- **推荐：** 30-50% 正常打印速度
- **轮廓：** 更慢（20-30mm/s）
- **填充：** 可稍快（40-60mm/s）

#### 首层挤出率
- **标准：** 105-110%（补偿压扁效果）
- **附着力差：** 可增加至115%
- **过挤出：** 减少至100-105%

### 6.3 热床表面处理

#### 清洁
- **日常：** 异丙醇擦拭
- **深度：** 洗洁精+热水
- **禁止：** 丙酮（损坏涂层）

#### 胶棒使用
**何时使用：**
- Engineering Plate打印任何材料
- PEI Plate打印PETG
- 附着力不足时

**使用技巧：**
- 薄薄一层即可
- 温床预热后涂抹
- 定期清理重新涂抹

### 6.4 附着力问题诊断

#### 翘边（Warping）
**原因：** 冷却收缩、床温不足、无封闭仓体
**解决：** 提高床温、使用封闭仓体、减少填充率

#### 首层太挤（Elephant Foot）
**原因：** Z偏移太小、首层温度太高
**解决：** 调整Z偏移、降低床温、减少首层挤出率

#### 附着力不足
**原因：** 床面不平、温度不足、表面污染
**解决：** 重新调平、提高床温、清洁表面、使用胶棒

## 7. 打印速度与质量权衡

速度和质量往往是矛盾的，需要根据应用需求平衡。

### 7.1 速度分类设置

#### 外轮廓（Outer Perimeter）
- **精细：** 20-40mm/s
- **标准：** 40-80mm/s
- **快速：** 80-120mm/s

**影响：** 表面质量、尺寸精度

#### 内轮廓（Inner Perimeter）
- **可比外轮廓快50-100%**
- **标准：** 60-150mm/s

#### 填充（Infill）
- **可以最快：** 100-300mm/s
- **限制：** 体积流速、材料特性

#### 顶底面
- **建议：** 比外轮廓慢20-50%
- **原因：** 保证表面质量

### 7.2 体积流速限制

**公式：** 体积流速 = 线宽 × 层高 × 速度

**材料限制：**
- **PLA：** 15-25 mm³/s
- **PETG：** 10-20 mm³/s
- **ABS：** 12-22 mm³/s
- **TPU：** 3-8 mm³/s

**超出后果：** 欠挤出、表面质量差、层间结合弱

### 7.3 速度策略

#### 质量优先（展示件）
```
外轮廓：30mm/s
内轮廓：60mm/s
填充：100mm/s
顶底面：25mm/s
```

#### 平衡设置（功能件）
```
外轮廓：60mm/s
内轮廓：120mm/s
填充：200mm/s
顶底面：40mm/s
```

#### 速度优先（原型）
```
外轮廓：80mm/s
内轮廓：160mm/s
填充：300mm/s
顶底面：60mm/s
```

### 7.4 加速度设置

**影响：** 打印时间、振动、表面质量

#### 保守设置（高质量）
- **外轮廓：** 500-1000 mm/s²
- **其他：** 1000-2000 mm/s²

#### 激进设置（快速）
- **外轮廓：** 1000-2000 mm/s²
- **其他：** 3000-5000 mm/s²

**注意：** 超出机械限制会导致振纹、漏步

## 8. 温度设置详解

温度是FDM打印最关键的参数之一，影响流动性、附着力、强度。

### 8.1 喷嘴温度影响

#### 温度过低
**现象：**
- 挤出不畅
- 层间结合差
- 表面粗糙
- 填充不足

**解决：** 提高5-10°C

#### 温度过高
**现象：**
- 过度流动
- 拉丝严重
- 尺寸不准
- 变色/降解

**解决：** 降低5-10°C

### 8.2 材料温度范围详解

#### PLA温度调节
- **基础：** 190-220°C
- **高速：** 210-240°C（提高流动性）
- **精细：** 190-210°C（减少流动）
- **PLA+：** 200-230°C
- **PLA-CF：** 210-240°C

#### PETG温度调节
- **标准：** 240-260°C
- **高速：** 250-270°C
- **透明：** 230-250°C（避免雾化）
- **PETG-CF：** 240-270°C

#### ABS温度调节
- **标准：** 240-260°C
- **高质量：** 250-280°C
- **快速：** 240-260°C

### 8.3 热床温度策略

#### 材料需求
**低温材料（PLA）：**
- 起始：60°C（增强附着）
- 持续：45°C（防止过软）

**中温材料（PETG）：**
- 全程：70-80°C

**高温材料（ABS/PA）：**
- 全程：90-120°C
- 需要封闭仓体配合

### 8.4 温度塔测试

**目的：** 找到材料最佳温度

**步骤：**
1. 设置温度范围（如190-240°C）
2. 每层递减5°C
3. 观察表面质量、层间结合、拉丝情况
4. 选择最佳温度区间

**评判标准：**
- 表面光滑度
- 层间结合强度
- 拉丝程度
- 尺寸精度

### 8.5 环境温度（仓体温度）

#### 开放式打印（PLA/PETG）
- **环境：** 20-30°C
- **注意：** 避免冷风直吹

#### 封闭式打印（ABS/PA）
- **目标仓温：** 40-80°C
- **PLA：** 需要开门/开盖散热
- **PETG：** 中等仓温（30-50°C）

### 故障排除快速索引

**附着力问题** → 检查床温、调平、清洁表面
**翘边变形** → 封闭仓体、提高床温、使用胶棒
**拉丝** → 降低温度、调节回抽
**层间分离** → 提高温度、降低速度
**表面粗糙** → 降低速度、优化温度
**尺寸不准** → 检查挤出倍率、温度

### 最终建议

1. **从简单开始：** PLA + 0.2层高 + 20%填充率
2. **逐步优化：** 先解决附着力，再调整质量和速度
3. **记录参数：** 成功的设置要做好记录
4. **材料特定：** 不同品牌同种材料可能需要不同参数
5. **环境考量：** 温度、湿度会影响打印效果
