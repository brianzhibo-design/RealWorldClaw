### 打印方向建议

- **上壳**：正面朝下（屏幕孔面贴床），无需支撑
- **下壳**：底面朝下，T型槽朝下贴床
- **屏幕压框**：平放
- **电池托盘**：底面朝下
- **PCB 铜柱**：法兰面朝下

## ✅ 可打印性检查

- [x] 所有壁厚 ≥ 1.2mm（实际 1.5-1.8mm）
- [x] 无悬空/桥接 > 45°
- [x] 卡扣容差 0.3mm（适合 FDM）
- [x] 螺丝柱 M2（ID 2.2mm，热熔螺母或自攻）
- [x] 总打印时间 < 2 小时
- [x] 单件均在 55×55×14mm 内，P2S 打印范围无压力
