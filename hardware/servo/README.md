# RWC Servo Module (2U: 40×20mm)

## 概述
16通道PWM舵机驱动模块，通过I2C控制，独立供电。

## 功能
- PCA9685 16通道PWM（25MHz晶振，I2C地址0x40）
- 4个SG90舵机3pin接口（引出CH0-CH3）
- 独立5V 2A供电输入（DC barrel jack 5.5×2.1mm）
- 不从RWC Bus取电，仅用Bus的I2C通信

## RWC Bus连接
| Pin | 功能 |
|-----|------|
| SDA | PCA9685 I2C数据 |
| SCL | PCA9685 I2C时钟 |
| GND | 公共地（Bus GND与舵机GND互联） |
| ID | DS28E05 1-Wire EEPROM |

## 舵机接口映射
| 接口 | PCA9685通道 | 用途建议 |
|------|-------------|---------|
| J1 | CH0 | 底座旋转 |
| J2 | CH1 | 大臂 |
| J3 | CH2 | 小臂 |
| J4 | CH3 | 爪子 |
