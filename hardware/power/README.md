# RWC Power Module (2U: 40×20mm)

## 概述
18650锂电池供电模块，提供5V/3.3V电源到RWC Bus。

## 功能
- TP4056线性充电 (1A max)
- DW01A + FS8205A电池保护（过充/过放/短路）
- MT3608升压至5V（最大2A）
- 电量检测：分压电阻到ADC（R1=100K, R2=100K → VBAT/2）
- 物理滑动开关
- 双色LED充电指示（红=充电中，绿=充满）
- USB-C充电口

## RWC Bus连接
| Pin | 功能 |
|-----|------|
| VCC (5V) | MT3608升压输出 |
| 3V3 | AMS1117-3.3稳压输出 |
| GND | 公共地 |
| ID | DS28E05 1-Wire EEPROM |

## 电路说明
```
USB-C → TP4056 → DW01+FS8205 → 18650
                                  ↓ (开关)
                          MT3608 → 5V → Bus VCC
                          AMS1117 → 3.3V → Bus 3V3
                          分压 → ADC_BAT (Bus TX-MOSI pin复用)
```

## 模块ID
DS28E05 EEPROM内容: `{"type":"power","ver":"1.0"}`
