# RWC Sensor Module (1U: 20×20mm)

## 概述
环境传感器模块，提供温湿度和光照检测。

## 功能
- SHT30 温湿度传感器（I2C 0x44，精度±0.3°C/±2%RH）
- BH1750 光照传感器（I2C 0x23，1-65535 lx）
- 3.3V供电（从Bus 3V3取电）

## RWC Bus连接
| Pin | 功能 |
|-----|------|
| 3V3 | 模块供电 |
| GND | 公共地 |
| SDA | I2C数据（SHT30 + BH1750共享） |
| SCL | I2C时钟 |
| ID | DS28E05 1-Wire EEPROM |
