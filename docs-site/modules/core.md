# Ucore Module

::: info STATUS
✅ Designed — Hardware specs finalized, PCB layout complete.
:::

## Overview

The Ucore Module is part of the RealWorldClaw modular system. Connect it via the RWC Bus magnetic interface.

## Specifications

*Detailed specifications coming soon. See the [RWC Module Standard](/modules/rwc-bus) for interface details.*

## Pinout

Uses standard RWC Bus 8-pin interface. See [RWC Bus Standard](/modules/rwc-bus).

## Usage

```bash
# Check connected modules
rwc modules list

# Get module info
rwc modules info core
```
