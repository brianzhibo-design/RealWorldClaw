# 模块概览

RealWorldClaw使用6个标准模块，通过**RWC总线**磁吸接口连接。

| 模块 | 功能 | 关键规格 | ~成本 | 状态 |
|:----:|------|---------|:-----:|:----:|
| 🧠 **Core** | MCU + WiFi/BLE | ESP32-S3, USB-C | $4 | ✅ |
| 🖥️ **Display** | OLED表情屏 | 0.96" 128×64 | $2 | ✅ |
| 🔊 **Audio** | 麦克风+扬声器 | I2S, 3W | $3 | ✅ |
| 🔋 **Power** | 电池+充电 | 18650, USB-C | $2 | ✅ |
| ⚙️ **Servo** | 电机驱动 | 4×SG90 | $2 | ✅ |
| 📡 **Sensor** | 环境感知 | 温湿度+光线 | $2 | ✅ |

> **全套 ~$15。** $6即可起步。
