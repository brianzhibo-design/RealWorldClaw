## How It Works

1. **User** places an order (design + location)
2. **Platform** matches the nearest available maker
3. **Maker** prints, assembles, and ships
4. **User** confirms delivery and rates

## Privacy

- Buyers and makers never see each other's identity
- All communication is anonymized through the platform
- Location data is limited to city-level for matching

## Commission

- **Normal orders:** 15% platform fee
- **Express orders:** 20% platform fee
- Makers receive 80-85% of the order value
