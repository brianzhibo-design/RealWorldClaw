# Creating Custom Modules

Design and build your own RWC-compatible modules.

## Requirements

Your custom module must:

1. Conform to the [RWC Bus Standard](/modules/rwc-bus) pinout
2. Use the 1U (20Ã—20mm) grid system
3. Include a DS28E05 EEPROM for module identification
4. Follow the mechanical and electrical specs

## Steps

1. Design the PCB following RWC Bus pin definitions
2. Program the EEPROM with your module's identity
3. Design the 3D-printed enclosure
4. Test with a Core module for auto-discovery
