# Clawbie V3 接线指南

## 蟹爪舵机（必须）

| 舵机线色 | 连接到 M5StickC Plus2 | 说明 |
|---------|----------------------|------|
| 🟠 橙色 (信号) | **GPIO 26** (G26) | PWM 信号 |
| 🔴 红色 (VCC)  | **5V** (5V OUT)    | 电源正极 |
| 🟤 棕色 (GND)  | **GND**            | 电源负极 |

## 旋转舵机（可选）

| 舵机线色 | 连接到 M5StickC Plus2 | 说明 |
|---------|----------------------|------|
| 🟠 橙色 (信号) | **GPIO 36** (G36) | PWM 信号 |
| 🔴 红色 (VCC)  | **5V** (5V OUT)    | 与蟹爪舵机共用 |
| 🟤 棕色 (GND)  | **GND**            | 与蟹爪舵机共用 |

## 接线图

```
M5StickC Plus2 (底部 Grove 口 / HAT 口)
┌──────────┐
│  G26 ────┼──→ 蟹爪舵机 信号 (橙)
│  G36 ────┼──→ 旋转舵机 信号 (橙) [可选]
│  5V  ────┼──→ 舵机 VCC (红) [并联]
│  GND ────┼──→ 舵机 GND (棕) [并联]
└──────────┘
```

## 注意事项

1. **供电：** M5StickC Plus2 的 5V 输出可驱动 1-2 个 SG90（空载 ~100mA）。夹重物时如果重启，考虑外部供电。
2. **走线：** 杜邦线从底座背面的走线槽穿入，保持正面整洁。
3. **舵机线序：** SG90 标准线序为 信号-VCC-GND（橙-红-棕），插反会烧！
4. **GPIO 36 注意：** 此引脚为输入专用引脚，但 ESP32 的 LED PWM 仍可用于输出。如遇问题改用 GPIO 0 或 GPIO 25。
