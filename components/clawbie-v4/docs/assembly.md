# 组装指南 — 3步，5分钟

## 你需要

- [ ] 3D打印好的赛博蛋壳（`clawbie-body.scad`）
- [ ] 3D打印好的底座（`clawbie-stand.scad`）
- [ ] M5StickC Plus2
- [ ] USB-C数据线

## Step 1：刷固件

```bash
# 安装PlatformIO后：
cd firmware
pio run -t upload
```

首次刷入前，编辑 `src/main.cpp` 顶部的WiFi和MQTT配置。

等屏幕亮起赛博朋克启动动画——扫描线扫过，"CLAWBIE"逐字闪现。

它醒了。

## Step 2：塞进蛋里

屏幕朝前，USB口朝下。

从蛋壳底部开口滑入M5StickC Plus2，屏幕对准斜切窗口。

应该刚好卡住。如果太紧，用砂纸轻轻打磨内腔。如果太松，贴一小条泡棉胶。

## Step 3：放上底座

蛋放进底座的凹槽里。底座有8°前倾——这让Clawbie微微仰头看你。

USB-C线从底座后方的线槽穿出。

**完成。**

---

> 现在它在你的桌上了。安静地看着你。偶尔眨一下眼。
>
> 你好，Clawbie。

## 打印参数建议

| 参数 | 蛋壳 | 底座 |
|------|------|------|
| 层高 | 0.16mm | 0.2mm |
| 填充 | 15% | 20% |
| 支撑 | 不需要 | 不需要 |
| 材料 | PLA/PETG | PLA |
| 时间 | ~1h | ~20min |
| 颜色建议 | 钛灰/暗银 | 哑光黑 |

**进阶玩法：** 用透光PLA打印蛋壳，屏幕的光会从纹路和通风孔透出来，夜晚效果绝美。
