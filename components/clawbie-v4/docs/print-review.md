## 关键设计修改

### 1. 蛋壳高度 55→58mm ⚠️ 严重
原始内腔高度 = 55 - 4(壁厚) = 51mm，M5StickC Plus2高54.2mm，**放不进去**。  
修改后内腔 = 58 - 4 = 54mm，配合0.3mm公差刚好。

### 2. 底部插入开口 🆕
原设计是封闭蛋壳，只有USB-C小孔(10×8mm)，M5StickC Plus2(25.5×13.2mm)无法装入。  
新增28×16mm底部矩形开口 + 1mm倒角边缘 + 内部导轨。

### 3. 装饰凹槽加大
FDM 0.4mm喷嘴的实际最小可见特征约1mm。原始0.6/0.8mm凹槽打印后几乎不可见。

### 4. 通风孔加大
1.2mm孔在FDM上会因孔径缩小(~0.3mm)变成<1mm，基本堵死。改为2mm。
