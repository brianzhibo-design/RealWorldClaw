# Clawbie V2 接线指南

## 基础版：无需接线 🎉

没错，你没看错。**零接线，零焊接，零工具。**

M5StickC Plus2 自带一切：屏幕、按钮、陀螺仪、蜂鸣器、电池、USB-C。

直接套壳开机。结束。

---

## 完整版：DHT22 温湿度模块（可选）

如果你想要精确的温湿度数据（而不是IMU芯片的粗略温度），接一个DHT22。

### 接线（3根杜邦线）

```
DHT22          M5StickC Plus2 (Grove端口/HAT引脚)
──────         ─────────────────────────────
VCC  ────────→ 3V3
GND  ────────→ GND
DATA ────────→ G26
```

### 说明

- 使用M5StickC Plus2顶部的HAT引脚排
- DHT22模块自带上拉电阻，无需额外元件
- 杜邦线直插即可，不用焊接
- DATA线建议用G26（GPIO26），固件默认配置

### 没有DHT22？

没关系！固件会自动使用MPU6886 IMU内置的温度传感器。精度差一些（±2°C），但够用。湿度显示会标记为"N/A"。
