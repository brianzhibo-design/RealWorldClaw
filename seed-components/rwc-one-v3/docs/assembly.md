# 🔧 Clawbie V3 组装指南

**总时间：10 分钟** | **难度：⭐ 超简单** | **工具：十字螺丝刀（M3）**

---

## 📦 你需要

- [ ] 3D 打印件 ×5（见下方）
- [ ] M5StickC Plus2 ×1
- [ ] SG90 舵机 ×1
- [ ] 杜邦线（母-母）×3
- [ ] M3×8 螺丝 ×4 + M3 螺母 ×4

## 🖨️ 打印清单

用 `models/` 下的 SCAD 文件生成 STL：

| 零件 | 文件 | 数量 | 打印时间 | 备注 |
|------|------|------|---------|------|
| 底座 | clawbie-v3-base.scad | 1 | ~60min | 正面朝上打印 |
| 左蟹爪 | clawbie-v3-claw.scad | 1 | ~20min | 平放打印 |
| 右蟹爪 | clawbie-v3-claw.scad | 1 | ~20min | 镜像件 |
| 推杆 | clawbie-v3-claw.scad | 2 | ~10min | 在同一文件的打印布局中 |
| 舵机摆臂 | clawbie-v3-claw.scad | 1 | ~10min | 连接舵机和推杆 |

**打印设置：** PLA, 0.2mm层高, 20%填充, 无支撑

---

## 🔨 组装步骤

### Step 1：装舵机到底座 (2min)

1. 将 SG90 舵机从底座顶部**卡入**舵机卡座
2. 听到"咔"一声 = 到位
3. 舵机线从背面走线槽穿出

### Step 2：组装蟹爪 (5min)

1. 将**舵机摆臂**套在舵机齿轮轴上（十字花）
2. 用 M3×8 螺丝将**左蟹爪**安装到底座的枢轴位，螺母在背面
3. 同样安装**右蟹爪**
4. 用 M3×8 螺丝将两根**推杆**分别连接：
   - 一端连舵机摆臂
   - 另一端连蟹爪臂的连杆孔
5. 手动转动舵机轴，确认蟹爪能顺畅开合

### Step 3：插入 M5StickC Plus2 (1min)

1. 屏幕朝前，竖直插入底座前方的插槽
2. USB-C 口朝下，对准底部开口

### Step 4：接线 (2min)

3 根杜邦线：

| 舵机线 | → M5 引脚 |
|--------|-----------|
| 橙 (信号) | G26 |
| 红 (VCC) | 5V |
| 棕 (GND) | GND |

### Step 5：烧固件

```bash
cd firmware
# 编辑 src/main.cpp 中的 WiFi 和 MQTT 配置
pio run -t upload
```
