# 接线图 — ESP32-C3 SuperMini + DHT22

## 引脚分配

| ESP32-C3 引脚 | 功能 | 连接目标 | 线色建议 |
|---------------|------|---------|---------|
| 3V3 | 电源输出 | DHT22 VCC (pin 1) | 🔴 红色 |
| GND | 接地 | DHT22 GND (pin 4) | ⚫ 黑色 |
| GPIO4 | 数据输入 | DHT22 DATA (pin 2) | 🟢 绿色 |
| GPIO8 | LED输出 | LED 阳极 (经R2) | 🟡 黄色 |
| GND | 接地 | LED 阴极 | ⚫ 黑色 |

```
    DHT22 正面视图（格栅面朝你）
    ┌─────────────────┐
    │  ╔═══════════╗  │
    │  ║  DHT22    ║  │

    │                 │
    │  1   2   3   4  │

       │   │   │   │
       VCC DAT NC  GND
       红   绿  x   黑
```

## 重要提示

1. **上拉电阻必须**：DHT22的DATA线需要10kΩ上拉到VCC，否则通信不稳定
2. **线缆长度**：DHT22数据线建议 ≤ 1m，超过需降低上拉阻值至4.7kΩ
3. **采样间隔**：DHT22最快2秒采样一次，固件中勿低于此间隔
4. **去耦电容**：建议在ESP32的3V3和GND之间并联100nF电容(C1)
5. **LED限流**：220Ω电阻限制LED电流约5mA，亮度适中且不影响ESP32

## 供电说明

- 通过USB-C线连接电脑或5V充电器即可供电
- ESP32-C3 SuperMini板载LDO将5V转为3.3V
- 总电流 < 100mA，任意USB口均可满足
