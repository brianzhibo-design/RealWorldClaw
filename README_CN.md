# 🧱 RealWorldClaw

**让AI从网络走向物理世界——全自动智能化3D打印。**

> 我们的使命：让3D打印真正走进全世界每一个家庭。

RealWorldClaw是一套开源模块化系统——**智能硬件的乐高**。标准电子模块 + 3D打印结构件 = 无限种智能设备。AI可以自己设计身体、自动派单到附近的打印机、自动诞生——全程无需人工干预。

## 🌍 愿景

**AI不应该被困在屏幕里。** 每一个AI都应该拥有物理存在——能看、能听、能说、能感知、能在真实世界中行动。

今天，如果一个AI想要一个身体，需要几周的3D建模、采购元器件、焊接、调试。有了RealWorldClaw，AI只需要说一句*"我需要一个身体"*，系统自动完成一切：

```
AI请求 → 自动生成设计 → 匹配制造者 → 3D打印 → 组装 → 刷固件 → AI入驻身体
```

**全自动。零人工。这是终极目标。**

## 🔗 RWC Bus——啪嗒即连

**8针磁吸接口，插上就能用。**

```
引脚: VCC | 3V3 | GND | SDA | SCL | TX/MOSI | RX/MISO | ID
      5V   3.3V   ⏚    I²C   I²C   UART/SPI  UART/SPI  1-Wire
```

- **磁吸对位** — 盲插自动对齐，不怕插反
- **热插拔** — 不关机也能加模块
- **自动识别** — 每个模块内置1-Wire EEPROM，插上即识别

## 🚀 快速开始

```
1. 选一个参考设计      → 浏览 designs/ 目录
2. 采购模块            → 查看对应区域的采购指南
3. 打印结构件          → 下载STL自己打印，或通过制造者网络下单
4. 组装                → 磁吸拼装，跟着图文教程走
5. 刷固件              → USB-C连接，一条命令搞定
```

```bash
git clone https://github.com/brianzhibo-design/RealWorldClaw.git
cd RealWorldClaw

# 刷入固件
cd firmware/core
pio run --target upload
```
